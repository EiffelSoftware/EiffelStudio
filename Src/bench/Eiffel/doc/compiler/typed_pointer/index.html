Typed pointer

Introduction

We are introducing TYPED_POINTER [G], a generic class whose content is:

indexing
	description: "Typed pointer. To be used for better interfacing with externals: C, C++, .NET."
	date: "$Date$"
	revision: "$Revision$"

class 
	TYPED_POINTER [G]

create {NONE}

convert
	frozen to_pointer: {POINTER}

feature -- Conversion

	frozen to_pointer: POINTER is
			-- Convert current to TYPED_POINTER.
		do
		end
	
feature -- Access

	frozen item: G is
			-- Content of pointer.
		do
		end
	
feature -- Settings

	frozen put (v: G) is
			-- Replace content of pointer by v.
		do
		ensure
			set: item = v
		end
	
end -- class TYPED_POINTER

The sole purpose of this class is to represent the type of the dollar expressions where the expression is not a routine.

Implementation

It is like pointer, except that at the C level we will generate something of type EIF_XX * where XX is either INTEGER_YY, DOUBLE, REAL, ..... On .NET it is a managed pointer of the given type.
