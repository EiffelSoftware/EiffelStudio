Security notes

  • When querying `has_entry', the user must have FileIOPermission (Read)

Unhandled exceptions

  • When querying `has_query' an exception (DirectoryNotFoundException) can be raised if the directory doesn't exist.
  • When querying `has_query' an exception (ArgumentException) can be raised if the directory name is invalid (empty, or contains invalid characters).
