4. Integrity checking


ISE reflection interface provides some other useful features relating to integrity checking:

  • `check_type_integrity (a_type: SYSTEM_TYPE)'
    -- Check that XML file and Eiffel class corresponding to `a_type' are matching.
    -- Make result available in `valid_type'.


  • `check_assembly_integrity_from_type (a_type: SYSTEM_TYPE)'
    -- Check integrity of all types of the assembly. -- Make result available in `valid_assembly'.


  • `check_assembly_integrity_from_info (a_version, a_culture, a_public_key: STRING)'
    -- Check integrity of all types of the assembly. -- Make result available in `valid_assembly'.


  • `check_assemblies_integrity'
    -- Check integrity of all assemblies -- Make result available in `valid_assemblies'.
