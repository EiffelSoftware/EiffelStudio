EiffelStore library content

EiffelStore main classes are grouped into 2 main layers:

  • The interface layer provides a high-level, unique interface for every DBMS.
  • The implementation layer customizes high-level routines to various DBMS handles.

Other cluster can facilitate your database management:

  • The generation cluster, with the EiffelStore wizard, generates a facilitated and dynamic interface with your database.
  • The dataview cluster lets you create a customized database GUI.
