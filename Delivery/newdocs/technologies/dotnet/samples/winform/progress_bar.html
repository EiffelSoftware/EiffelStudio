Running

After launching the application, you will see a window displayed with a similar appearance to the one above. On the left of the form you have the PROGRESS_BAR control, and on the right you have some controls to configure the PROGRESS_BAR.

 

Under the Hood

The application shows how to configure a PROGRESS_BAR.

The feature on_load is redefined and launch a thread (THREAD_START) that simulate the progression of a process. 

This sample contains the following class:

  • PROGRESS_BAR
  • GROUP_BOX
  • TRACK_BAR
  • COMBO_BOX
  • LABEL
  • THREAD_START

Notes

This sample is translated from the example located in the QuickStart\winforms\samples\controlReference\ProgressBar subdirectory of the .NET Framework SDK samples directory of Microsoft Visual Studio .NET.
