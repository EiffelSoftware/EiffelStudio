Generating executables

There are several ways to compile an executable with EiffelStudio.

Compilation modes include the following: melting, freezing and finalizing. To learn more about the compilation semantics in EiffelStudio, see the compiler reference.

There is a command for all compilation modes in the Project menu, under the entries Melt, Freeze and Finalize. There is also a Quick compile command, that only melts the classes that have been edited in EiffelStudio since the last compilation. You may want to use it as long as no classes are edited outside EiffelStudio.

These commands are also available in the Project toolbar , , , .

To solve the errors that occur during the Eiffel compilation, use the compilation error wizard. For errors occurring during the C compilation, check that the installation is not corrupted (all environment variables should be set, especially ISE_C_COMPILER and ISE_PLATFORM), and refer to your C compiler documentation for more information.
