Object editor

An object editor is a tool which allows you to edit the properties of a selected object. Common properties include minimum width and height, color, text etc etc.

An object editor can only reference one object at once, and when not empty, it is considered to be "targeted" to the object whose properties it references.

Docked object editor

The docked object editor is displayed to the right hand side of the main Build window. This editor is always displayed and is permanently linked to the layout constructor. This means that whenever the selected object changes in the layout constructor, the docked object editor becomes targeted to this object.

Floating object editors

These object editors appear in their own window, and you may open as many of these as you wish. To open a new floating object editor, drop an object on to the tool bar button displayed in the main toolbar.
