Opening a compiled project

Whenever a project has been compiled successfully, files are created in the project directory. These files include the .epr file and the EIFGEN directory. They contain all the information produced by the compiler. After exiting EiffelStudio, it is possible to open again a project that has been compiled by indicating the location of the .epr file to the system.

To do this:

  • In the startup dialog, select Open compiled project. If the .epr file you want to open is not in the list, click Browse and indicate the location of the .epr file.

  • Or, click the File/Open Project... menu and indicate the location of the .epr file.
