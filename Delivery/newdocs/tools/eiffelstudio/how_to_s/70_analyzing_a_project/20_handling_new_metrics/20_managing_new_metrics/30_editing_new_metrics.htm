Editing new metrics

To edit a metric, you must have already clicked on in the "Metric" tab of the context tool.

  1. Select a metric
  2. Click on the "Edit" button or double click on the metric.
    The interface previously used to define new metrics will pop up, filled regarding the selected metric's definition. For e.g. editing the selected metric above leads to:


  3. If you make some changes, "Save" will be sensitive so that you can register your modifications.
  4. You can also define other new metrics if wished
  5. Click on "OK" to save your changes if any and exit. Click on "Cancel" to exit editing.
  6. If you have introduced some changes, you have to save again on the main interface.
  7. Click on "Exit" to hide the interface.
