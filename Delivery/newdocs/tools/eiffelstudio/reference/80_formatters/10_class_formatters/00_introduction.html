Introduction

Class formats are all available through the Class tab of the HYPERLINK(context tool) context tool, and for some of them directly with the HYPERLINK(editor introduction) editor.
Classes can be displayed in the following formats:
  • Basic text form
  • Clickable form
  • Flat form
  • Contract form
  • Interface
  • Ancestors
  • Descendants
  • Clients
  • Suppliers
  • Attributes
  • Routines
  • Deferred features
  • Once routines and constants
  • External features
  • Exported features
