The open assembly dialog


The import tool lists the assemblies from the Global Assembly Cache. Nevertherless, ISE Assembly Manager only requires that an assembly is signed to import it. Therefore, the import tool gives you access to an open assembly dialog (Menu `File', item `Open Assembly').
The open assembly dialog is a Windows open file dialog, with default extensions `*.dll; *.exe', which are the only possible extensions for a .NET assembly.

When you click on `Open', ISE Assembly Manager will first check that the selected assembly is signed. If it is a signed assembly, it will open the import dialog, which will help you import the assembly. Otherwise, a message box will appear, inviting you to sign the assembly before importing it with the assembly manager.
