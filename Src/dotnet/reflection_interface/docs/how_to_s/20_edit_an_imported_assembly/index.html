How to edit an imported assembly?

  1. Go to the Imported Assembly Viewer.
  2. Click on the name of the assembly.
  3. Select Edit from the Tools menu or click on the corresponding toolbar button .
  4. In the Assembly View, click on the type name (the Eiffel name) to open the corresponding Type View.
  5. Click on the feature name or the argument name you wish to edit.

Note: Only immediate features (i.e. features that are not inherited) and their arguments names are editable. To rename a feature which is inherited (or an argument whose related feature is inherited), open the Type View corresponding to the class where the feature is first declared.
