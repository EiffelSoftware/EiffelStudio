Customizing class text views

It is possible in EiffelStudio to customize some properties of class text views (other than basic texts) displayed in the editor or class and feature tabs of the context tool. Besides graphical appearance, you may modify:
  • the order of feature clauses.
  • the type of callers and suppliers displayed in the corresponding views.
  • the views that are selected by default.
  • the indexing clauses that should not be displayed.
These properties are set in the preferences. You can modify them in the preferences window. To open this window, select Preferences... in the Tool menu. Then select the "Context tool" sub-category of the "Tools" category in the preferences tree. Click then on a preference on the right to modify its value.

Click on save and exit to complete the changes.
