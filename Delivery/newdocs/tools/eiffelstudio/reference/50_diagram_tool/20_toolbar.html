Diagram toolbar

The customizable toolbar of the diagram tool can contain these buttons:

  • Create a new class
    Pick from this button and drop on the diagram to add a new class.
  • Delete
    Drop items in this hole to remove them from the system.
  • Create new inheritance links
    Selecting this button allows you to create inheritance links by picking from one class and dropping on the other.
  • Create new client-supplier links
    Selecting this button allows you to create client-supplier links by picking from one class and dropping on the other.
  • Create new aggregate client-supplier links
    Selecting this button allows you to create aggregate client-supplier links by picking from one class and dropping on the other.
  • Show/hide client-supplier links
    Toggle visibility of all client-supplier links in the view.
  • Show/hide inheritance links
    Toggle visibility of all inheritance links in the view.
  • Show/hide labels
    Toggle visibility of all client link labels in the view.
  • Remove figure
    Drop figures in this hole that you want to remove from the view only.
  • Change color
    Drop class bubbles in this hole to change their color.
  • Change class name and generics
    Drop class bubbles in this hole if you want to change a name and/or formal generics.
  • Put handles on a link
    Click on this button to toggle right angles on all links in the view. Drop a link on it to bring up the link tool for that link.
  • Select depth of relations
    Click to bring up the Select depth dialog.
  • Include all classes of cluster
    Drop clusters in this hole to show all classes in the system in this view.
  • Undo last action
    Click to undo the last action.
  • History tool
    Click to open the History tool.
  • Redo last action
    Click to redo the last undone action.
  • Zoom in
    Enlarge the diagram.
  • Zoom out
    Shrink the diagram.
  • Delete current view
    Click to remove the current view from the list of views for current class or cluster.
  • Export diagram to PNG
    Click to generate a .PNG file from current view.
