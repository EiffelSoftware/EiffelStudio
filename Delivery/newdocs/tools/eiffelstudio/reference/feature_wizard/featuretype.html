Feature types

With the feature wizard you can create procedures, functions or attributes. With every feature type, the window has a different layout. The common components of the layout are for all feature types:

  • Feature clause selection
  • Feature name field
  • Header comment field

Attribute

When clicking attribute, the wizard changes to the attribute layout. It has the following components:
  • Type selection
  • Invariant field
  • Set-procedure checkbox

Procedure

The procedure layout has:
  • Argument list
  • Precondition field
  • Local variable field
  • Selection for: normal, once, deferred or external routine
  • Postcondition field

Function

The layout of a function is very similar to that of a procedure; the only difference is that the function layout adds:
  • Type selection
