Vision2 Primitives

All vision2 primitives inherit EV_PRIMITIVE.

What is a primitive?

A primitive is a Vision2 widget that may not contain other widgets. Primitives may be placed in containers, but no widgets may be contained in a primitive. Some examples of primitives are EV_BUTTON and EV_LABEL.

Features

All primitives inherit from EV_TOOLTIPABLE and therefore can have tooltips assigned to them. Each descendent of EV_PRIMITIVE will typically have many available features, specific to their type. For example, EV_SEPARATOR has no extra features, but EV_LABEL has features for modifying the current font and also the text to de displayed.

Items

Although no primitive can contain another widget, certain primitives can contain items. The following primitives can contain items:
  • EV_TOOL_BAR can contain items of type EV_TOOL_BAR_ITEM
  • EV_MULTI_COLUMN_LIST can contain items of type EV_MULTI_COLUMN_LIST_ROW
  • EV_LIST can contain items of type EV_LIST_ITEM
  • EV_COMBO_BOX can contain items of type EV_LIST_ITEM
  • EV_TREE can contain items of type EV_TREE_NODE
