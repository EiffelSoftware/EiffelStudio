EV_ACCELERATOR

What is an accelerator?

An EV_ACCELERATOR provides a means of associating a keyboard shortcut to an EV_TITLED_WINDOW. Facilities provided by the class allow you to set the key combination which is any valid EV_KEY with a combination of the shift, alt and control keys.

Associating with an EV_TITLED_WINDOW

EV_TITLED_WINDOW has a feature accelerators of type EV_ACCELERATOR_LIST and to connect your accelerator to the titled window, it must be inserted into this list. Whenever a key combination matching that of an accelerator within accelerators is recieved by the window, the actions of that accelerator are fired.

Note:Accelerator key combinations are only recieved by the currently focused window.
