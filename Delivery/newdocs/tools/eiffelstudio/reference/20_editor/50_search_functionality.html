Search functionality

[BENCH] editor provides a search functionality. To search an expression in the edited text, you can either use the search tool (see HYPERLINK(search_tool)) or use a set of keyboard accelerators available directly from the editor. From the editor, you can:

  • Search the next occurence of the last searched pattern.
    To do so, press "F3". You can change this key binding in the preferences (HYPERLINK (editor_preferences)). If you press "F3" as some text is selected, the editor will highlight the next occurrence of this text, if any. You can change this behavior in the preferences HYPERLINK(editor_preferences).

  • Search the previous occurence of the last searched pattern.
    To do so, press "Shift+F3". You can change this key binding in the preferences (HYPERLINK (editor_preferences)) too.
