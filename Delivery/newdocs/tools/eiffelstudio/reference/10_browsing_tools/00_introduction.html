Introduction

The browsing tools are all the tools that are in the left panel of the development windows as well as the address bars. They help find an item in the system, such as a class, a cluster, a feature, some text, ... They include the address bars, the cluster tool, the feature tool, the favorites tool, the windows tool and the search tool.

The main address bar may be shown or hidden by using the View/Toolbars/Address Bar menu, or by right-clicking on the right of the toolbar and clicking Address bar in the popped up context menu.

All left-hand tools can be hidden or shown using their associated toolbar button, or through the View/Explorer Bar menu. They also all have a common mini-toolbar that handles window-like features:

  • Minimize ( ) iconifies the tool, and leaves only its toolbar visible at the bottom of the panel.
  • Maximize ( ) enlarges the tool so that it occupies the whole space of the development window.
  • Restore ( ), available whenever a tool has been minimized or maximized, puts it back to its previous state.
  • Close ( ) discards the tool from the panel, as if its associated toolbar button had been pressed.

Note: No more than three tools can be displayed at the same time in the left panel. Trying to add more will automatically minimize one of the displayed tools, in order to make room for the new tool.
