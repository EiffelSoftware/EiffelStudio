Managing new metrics

After having defined some new metrics, it is possible to reorder them, to delete the useless ones, or to edit them. It is also possible to import a set of metrics defined in another file.

In order to achieve any of the preceding operations, click on . The following dialog will help you for:

  • Reordering new metrics
  • Deleting new metrics
  • Editing new metrics
  • Importing new metrics.
