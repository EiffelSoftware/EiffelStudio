WEL_UP_DOWN_CONTROL
A WEL_UP_DOWN_CONTROL consists of a pair of arrows which are used to increment or decrement a value. Using set_buddy_window, a window of type WEL_WINDOW can be linked to this control, and is commonly used to display the current value.
