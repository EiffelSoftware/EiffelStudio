Evaluating a measure

To evaluate a metric over a scope:

  1. Select the object you want to evaluate the metric on (a feature, a class or a cluster. System is compulsory the running one).
  2. Select the scope type from the scope combo box (you can either choose the scope type of the previously selected object or a greater scope type in the sense of the following order relation: feature < class < cluster < system < archive). 



    Note: If you choose "Archive" as scope type, a dialog will ask for opening an archive file to compute the metric over. Select then an archive file.
  3. Select a metric from the metric menu.


  4. Click on the calculate button.

The measure is displayed in the provided text field below commands, see red rectangle in the following example:

In some cases, you can get more information about the evaluated metric. See detailing a measure.

Note: - Scope type list is either shrunk or extended regarding selected object (only greater scope types appear).
          - All metrics are not available for any scope type.

Caution: When calculated, a measure is not saved yet!
