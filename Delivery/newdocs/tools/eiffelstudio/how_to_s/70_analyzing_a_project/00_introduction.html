How to analyze a project

The metric tab in the context tool allows project analyzing. Built on a suitable metric theory, this tool provides a set of elementary metrics that can be evaluated to test projects design and implementation.

The metric tool provides the following interface to evaluate measures, save them or delete some of them when no longer needed.

See section:

  • Computing a measure

This tool also allows users to define their own metrics. Indeed, users may have specific requirements and the set of metrics provided by default could be insufficient. See section:

  • Handling new metrics
