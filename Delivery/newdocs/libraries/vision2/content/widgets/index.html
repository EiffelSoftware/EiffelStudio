Vision2 Widgets

A widget is a basic building block of a Vision2 application. In order to build a Vision2 application, you combine widgets to produce the desired GUI. All widgets inherit EV_WIDGET which provides many basic features such as show, hide, set_minimum_width and set_minimum_height. EV_WIDGET and therefore every Vision2 widget also inherits many common properties.

Within Vision2, widgets have been classified into three different groups:

  • Primitives
  • Containers
  • Common_dialogs
