Defining scope comparison metric

To define a scope comparison metric, you must have already clicked on of the "Metric" tab in the context tool. Select then the "Scope Ratio" tab of the "New metric" dialog:

  1. Enter a name in the "New metric name" field.
  2. Select a metric from the "Metric" combo box. 
  3. Select a scope for numerator in the "Scope" combo box of the "Numerator" frame to change the numerator in the "Ratio" field
  4. Select a scope for denominator in the "Scope" combo box of the "Denominator" frame to change the denominator in the "Ratio" field. 
  5. If you want to change again either metric, numerator or denominator repeat operations 2., 3. or 4.as many times as needed.
  6. If you want to display measures, when evaluating this metric, in a percentage style, select "Display as percentage".
  7. If you want to save the metric definition without exiting the interface, click on "Save", otherwise, click on "OK" to save and exit.
  8. If you want to define another scope comparison metric, click on  to reset.

Note: - The formula field below the metric combo box gives the selected metric's definition.
          - The ratio field is filled in by default, you have just to change metric, numerator and denominator as needed.
