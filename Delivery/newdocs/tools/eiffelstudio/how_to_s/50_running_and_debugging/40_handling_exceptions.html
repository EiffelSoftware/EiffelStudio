Handling exceptions

It is possible to raise and catch exceptions in Eiffel. Catching exceptions is done by using the rescue keyword. HYPERLINK(Eiffel reference). The EXCEPTIONS HYPERLINK(base reference EXCEPTIONS) class provides helper features to analyze the caught exception and handle it.

The EXCEPTIONS class also provides ways to raise exception, via its feature `raise' HYPERLINK(base reference EXCEPTIONS::raise).

When an exception is raised while the application is being debugged, the application stops immediately and the debugger displays the context in which the exception occurred, whether or not the exception is rescued.
