VALIDITY VTBT

ETL2 p.210 and ETR p.47:
----------------------------------------------------------------------
A Bit_type declaration is valid if and only of its Constant is of
type INTEGER, and has a positive value.
----------------------------------------------------------------------

----------------------------------------------------------------------

value. Validity VTBT is violated.
