Adding a cluster

It is possible to create new clusters from EiffelStudio. A dialog window has been designed to help you doing so. There are three ways to make this window appear. You can:
  • Click on in the standard toolbar.
  • Click on in the Cluster tool.
  • Select New cluster in the Tools menu.
This window displays a list of all the clusters in the system. Select an item in this list to set the parent of the new cluster and type its name in the text field above. Then, click then on the Create button to complete the process.

To import an existing cluster, in the Project Settings dialog, go to the Clusters tab and click Add. Give the newly added cluster the name you want and give it the path of the existing cluster.
