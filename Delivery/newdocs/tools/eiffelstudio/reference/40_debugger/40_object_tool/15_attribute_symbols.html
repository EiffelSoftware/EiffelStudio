Attribute symbols

In both trees of the object tool, objects are displayed. Two categories are made for each objects: its attributes (grouped under the `attributes' folder ( )) and its once routines (grouped under the `once routines' folder ( )).

All follow the following symbol scheme to describe the kind of each attribute:

  • An attribute of a basic type, such as INTEGER, DOUBLE,...
  • An attribute that is a reference to an object
  • A Void reference
  • An object that is expanded
