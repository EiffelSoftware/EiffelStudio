Renaming a class

Renaming a class can sometimes be tedious. It is necessary to update clients and ancestors of the renamed class, and they may be numerous. EiffelStudio offers a simpler way to remedy this problem.

The solution is provided by the Diagram tool: to rename a class, center the context tool on it and display the diagram. Then pick the class (from anywhere in EiffelStudio) and drop the pebble on . A dialog window will then appear. It will allow you to set the new name. It will also propose three options : Change name locally, Global search/replace of name in compiled classes and Global search/replace of name in entire universe. The first option is equivalent to modifying manually the class name in its text. The second option will rename the class and update its clients and ancestors. The third option will do the same as the second and will also update classes that are in the universe but not in the system.
Changes are actually made when you press OK.
