Gobo Eiffel Xace

Gobo Eiffel Xace, or gexace for short, is a tool which can convert Xace files to Eiffel compiler dependent configuration files. Xace, or XML Assembly of Classes in Eiffel, is an Eiffel compiler independent way to specify information needed to compile an Eiffel program. When it was first developped, gexace was able to convert Xace files to ISE's and HACT's Ace files, Visual Eiffel's ESD files and SmartEiffel loadpath files. Nowadays, only ISE's ECF files are supported. gexace can also be used to process Xace files in order to include clusters of mounted libraries directly in the resulting Xace file, or to filter out some parts of the Xace file which don't satisfy some conditions.
