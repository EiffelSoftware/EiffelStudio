Deleting measures

To delete a set of measures, you must have saved some of them previously. Recorded measures are displayed in the multi column list. Here is an example of a non-empty multi column list:

  1. Select a row (or a set of rows using SHIFT or CTRL keys) from the multi column list.
  2. Click on the delete button. Selected row (s) will disappear from both multi column list and from the recording file located in the "Metrics" subfolder of the current project.

Caution: Delete operation is not undoable!
