Updating an archive

After recompiling a project, measures are liable to have changed, and then, the archive created for this file -if any- is probably out of date. To update an archive, you mist have already clicked on in the "Metric" tab of the context tool.

  1. In the "Update archive" frame, click on "Browse".
  2. Select the archive file of the current system.
