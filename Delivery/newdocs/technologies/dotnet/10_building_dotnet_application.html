Building a .NET application

Eiffel.NET has some specific functionality meant to leverage necessary aspects of the .NET Framework.

For that reason, the project settings for Eiffel.NET introduces new options in the .NET tab of the project settings. These options include:

  • Specifying whether the generated assembly should be an EXE or a DLL.
  • Choosing between generating verifiable or non verifiable IL code. Non verifiable IL code executes faster but requires high trust settings.

The Eiffel.NET compiler generates a single assembly whose name is the name of the system as given in the project settings.
