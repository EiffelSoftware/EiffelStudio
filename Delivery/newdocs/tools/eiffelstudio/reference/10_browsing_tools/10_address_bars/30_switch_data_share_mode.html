Change data share mode

Located in the View menu, named either Isolate context tool or Merge context tool, this command allows you to switch between the two possible addressing modes. It is also accessible through the main toolbar ( ).

Note that this command only applies to the development window where it is invoked, and this setting is not automatically saved when exiting EiffelStudio.
