Introduction

The object tool is a debugging tool that displays extended dynamic information concerning one or several objects while debugging an application. This information includes the value of all their attributes, as well as the values of the once functions that have already been called.

It is mainly composed of two trees, the local view and the object view. In the toolbar are located two commands that facilitate browsing the trees: the remove object command and the set slice size command . The third command can give an expanded display of the string relative to an object, by dropping the object onto the button. Use it if an object has a very long string representation or a string representation that contains carriage returns, for example.
