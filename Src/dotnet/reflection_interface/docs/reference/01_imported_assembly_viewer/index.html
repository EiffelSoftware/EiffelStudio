The Imported Assembly Viewer

The imported assembly viewer is the main window of the Assembly Manager. It is the first to appear when you start the Assembly Manager. It gives access to all the facilities provided by the Assembly Manager.

The menu

The `File' menu

The `File' menu includes only one item: Exit.

The `View' menu

The first six items command which columns are displayed in the table. They correspond respectively to the six first toolbar buttons .

By default, only the assembly name and the path to the corresponding Eiffel sources are visible. You can also choose to view the assembly version, culture and public key or assembly dependencies.
