Main address bar

The main address bar is located in the top part of windows, between the two main toolbars (when they are displayed).

SCREENSHOT(Main address bar)

It has several components:

  • History back and forth arrows and
  • Fields where the current class or feature name is displayed
  • Format icons to change the format used by the editor , , , and

It is possible to type a class or feature name to center the editor on it. Typing a class name that does not exist in the set of known classes pops up a dialog to create a class of that name.

Tip: Wildcards (* and ?) are available in these fields.

Tip: The history is also accessible in the View/Go to menu.
