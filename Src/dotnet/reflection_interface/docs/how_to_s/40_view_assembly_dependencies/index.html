How to view assembly dependencies?


  1. ISE Assembly Manager gives you the possibility to view dependencies of an assembly, no matter if it is imported or not. Therefore you can view the dependencies from the imported assembly viewer and from the import tool.

  2. Click on the name of the assembly you want to see the dependencies (in the array).

  3. Then you have the choice between clicking on the toolbar button and selecting the menu item `Dependency Viewer' in the `Tools' menu. In both cases the dependency viewer will appear.
