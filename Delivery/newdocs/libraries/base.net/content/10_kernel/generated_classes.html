Generated classes

The following classes are generated from Microsoft .NET libraries. They represent the basic and most commonly used types for Eiffel.NET needs. Microsoft puts a detailed documentation at disposal for every class. They are all gathered under the "System" namespace of the reference chapter.
However, .NET names may differ from the Eiffel.NET ones. Indeed, the Eiffel language also has basic structures equivalent to the .NET ones, and then, in order to have similar class and feature names in Eiffel.NET, .NET classes and features have been sometimes renamed to be closer to the Eiffel ones.
This section gives the names' correspondences between .NET and Eiffel.NET structures for the following classes:
