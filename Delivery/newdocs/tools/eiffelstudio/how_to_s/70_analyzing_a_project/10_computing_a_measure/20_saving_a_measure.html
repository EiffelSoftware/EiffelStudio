Saving a measure

To save the result of evaluation of a metric over a scope:

  1. Evaluate a metric over a scope.
  2. You can enter a name in the name field, otherwise, the metric tool generates automatically a name for the saved measure.
  3. Click on the add button. (link to buttons).

The result is added to the multi column list, with a icon, meaning that the measure is related to the last compilation of the project.

If you are saving measures for the first time, a file will be automatically generated to store measures. It is located in the "Metrics" subfolder of current project. Otherwise, this file is updated.

A dialog reminds you of the path of the file in which measures are stored.

Note: After recompiling the project, all icons are changed to to inform that measures could have changed since project has been recompiled. You can update your measures.
