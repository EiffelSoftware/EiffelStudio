General
	cluster: vision2.interface.properties
	description: 
		"Facilities for pick and drop mechanism.
		Decendents can act both as pick and drop sources and as targets.
		When the user picks a `pebble' from a source and drops on a target,the `drop_actions' of the target receive the `pebble' as input.
		The user interface can be either pick and drop or drag and drop,selected by `set_pick_and_drop' and `set_drag_and_drop'."
