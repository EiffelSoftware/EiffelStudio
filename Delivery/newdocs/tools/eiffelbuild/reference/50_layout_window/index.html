Display Window

The Display window provides a view of the objects in your system, but unlike the layout constructor and builder window, the objects cannot be manipulated.

This window may either be shown or hidden (default). To change between these two states, click on the main toolbar, or select Show/Hide display window from the View menu.

This view is provided so that you can really see what your project will look like when it has been generated and is running as a stand alone EiffelVision2 system.

All properties that have been set on the objects in your project are set in the EiffelVision2 control matching its type.
