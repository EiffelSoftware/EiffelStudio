Key shortcuts

Many operations in EiffelStudio do not require the use of the mouse. The same effect can be achieved through the keyboard only.

This page gathers all the keyboard shortcuts available in the environment. They are sorted in several categories, depending on when or where they apply.

Editor shortcuts

Key shortcut Purpose
- clipboard shortcuts -
Ctrl+C Copy : Copy selected text into the clipboard.
Ctrl+Ins Equivalent to Ctrl+C.
Ctrl+X Cut: Copy selected text into the clipboard and remove it.
Shift+Del Equivalent to Ctrl+X.
Ctrl+V Paste: Insert clipboard content into the text.
Shift+Ins Equivalent to Ctrl+V.
- undo/redo shortcuts -
Ctrl+Z Undo last action.
Ctrl+Y Redo last undone action.
- search shortcuts -
Ctrl+F Display the Search tool and give the focus to the "Search for..." text field.
This shortcut may be customized.
Ctrl+H Display the Search tool and give the focus to the "Replace with..." text field.
This shortcut may be customized.
F3 Search the next occurrence of currently selected text or, if there is no selection, of the last searched expression.
This shortcut may be customized.
Shift+F3 Search the previous occurrence the last searched expression.
This shortcut may be customized.
- shortcuts to modify selected text -
Tab Indent selection (only if several lines are selected).
Shift+Tab Unindent selected lines or current line if there is no selection.
Ctrl+K Comment selected lines or current line if there is no selection.
Ctrl+Shift+K Uncomment selected line or current line if there is no selection.
A line is uncommented only if '--' is at its beginning.
Ctrl+I Embed selection (or current line if there is no selection) in an "if...then...end" structure.
Ctrl+D Embed selection (or current line if there is no selection) in an "debug...end" structure.
- other shortcuts -
Ctrl+A Select the entire text.
Ctrl+Space Auto-complete current word.
This shortcut mau be customized.

Note: Most of these shortcuts can be used in the class view and the feature view of the class tool too.
In fact, all the operations that do not imply changes in the text are possible.
