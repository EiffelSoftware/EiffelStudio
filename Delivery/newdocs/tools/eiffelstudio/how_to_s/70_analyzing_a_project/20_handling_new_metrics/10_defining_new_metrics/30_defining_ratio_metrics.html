Defining ratio metrics

To define a ratio metric, you must have already clicked on of the "Metric" tab in the context tool. Select then the "Metric Ratio" tab of the "New metric" dialog:

  1. Enter a name in the "New metric name" field.
  2. Select a metric for numerator in the "Metric" combo box of the "Numerator" frame to change the numerator in the "Ratio" field. 
  3. Select a metric for denominator in the "Metric" combo box of the "Denominator" frame to change the denominator in the "Ratio" field. 
  4. If you want to change again either numerator or denominator repeat operations 2. or 3. as many times as needed.
  5. If you want to display measures, when evaluating this metric, in a percentage style, select "Display as percentage".
  6. If you want to save the metric definition without exiting the interface, click on "Save", otherwise, click on "OK" to save and exit.
  7. If you want to define another ratio metric, click on  to reset.

 

Note: - The formula field below each metric combo box gives the selected metric's definition.
          - The ratio field is filled in by default, you have just to change numerator and denominator as needed.
