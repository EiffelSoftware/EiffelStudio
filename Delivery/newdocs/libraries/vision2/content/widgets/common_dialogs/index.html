Vision2 dialogs

This cluster contains all the dialogs provided by vision2.

Standard dialogs

A standard dialog is an interface to a native operating system dialog. Operating systems provide a standard set of dialogs in order to provide a consistent user interface that lets users carry out common tasks without having to learn new techniques with different applications. Every standard dialog provided by vision2 is a descendent of EV_STANDARD_DIALOG, and the following such dialogs are available:

  • EV_COLOR_DIALOG
  • EV_DIRECTORY_DIALOG
  • EV_FILE_OPEN_DIALOG
  • EV_FILE_SAVE_DIALOG
  • EV_FONT_DIALOG
  • EV_PRINT_DIALOG

Creating custom dialogs

If you wish to provide a custom dialog in your application, EV_DIALOG has been provided to facilitate this.
