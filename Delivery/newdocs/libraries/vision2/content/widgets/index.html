Vision2 Widgets

A widget is a basic building block of a Vision2 application. In order to build a Vision2 application, you combine widgets to produce the desired GUI. All widgets inherit EV_WIDGET which provides many basic features such as show, hide, set_minimum_width and set_minimum_height. EV_WIDGET and therefore every Vision2 widget also inherits many common properties.

Within Vision2, widgets have been classified into three different groups:

  • Primitives -- These are elements of the user interface that are mainly responsible for interaction with the user, such as an EV_BUTTON.
  • Containers -- These are used to contain other widgets and position them in a certain way, such as an EV_VERTICAL_BOX that stacks its children one by one vertically
  • Common_dialogs -- These are Operating System dependant popup dialog windows used for interacting with the user for tasks such as opening a file (EV_FILE_OPEN_DIALOG)
