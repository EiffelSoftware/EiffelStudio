.NET Wizard guided tour


  1. First Step: Launch the .NET Wizard - Welcome !

  2. Second Step: Choose your project name and the working directory.

  3. Third Step: Choose the type of your application (.exe or .dll) and the names of the root class and its creation routine.

  4. Fourth Step: Select the .NET assemblies.

  5. Final Step: Check your settings summary and launch the code generation (and compilation).
