Description

Version: 1.0

Availability: Enterprise Edition

The new .NET Wizard aims at helping you to build your first Eiffel Sharp applications. In a very simple way, it will ask for your project settings and then automatically generate the Ace file (the control file) and the project root class (the entry point of the application) for you.

The wizard also gives you access to ISE Assembly Manager to enable you to import other external assemblies (if you need an assembly that is not imported yet to build your application).

Happy Eiffel Sharp developing with the new .NET Wizard !
