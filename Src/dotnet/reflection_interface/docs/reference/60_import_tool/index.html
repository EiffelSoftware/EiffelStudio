The Import Tool


The import tool almost looks like the imported assembly viewer (the main screen of ISE Assembly Manager). But instead of listing the imported assemblies (in the Eiffel assembly cache), it lists the shared assemblies (the assemblies in the Global Assembly Cache). Therefore the import tool mainly aims at enabling you to import signed assemblies to the Eiffel assembly cache. This reference aims at showing you all possibilities provided by the import tool.


The menu

The `File' menu

  • Open Assembly

    • This menu item (as well as the open assembly toolbar menu: ) gives you the possibility to import signed assemblies, which are not in the Global Assembly Cache by opening the open assembly dialog.

    See also: How to import a signed assembly to the Eiffel assembly cache?

  • Exit

    • The `File' menu provides you with another way to close ISE Assembly Manager by clicking on the `Exit' menu item.

The `View' menu

The first five menu items enable you to show new columns in the array. By default, only the assembly name is visible. But you may want to see also the assembly version, culture and public key or assembly dependencies. That is the aim of these five items. The first five toolbar buttons give you the same opportunity:
The last two items of the `View' menu are only a faster way to show all columns or come back to the initial layout.

Note that when you click on `Show dependencies', a new column will be added to the initial array. But it will only show the names of dependencies. If you want some more precise information about them, just select `Dependency Viewer' in the `Tools' menu to open the dependency viewer and have a more complete view of each dependency.

The `Tools' menu

  • Dependency Viewer

    • This item (as well as the following toolbar button:) gives you the ability to launch the dependency viewer, which lists all dependencies of the selected assembly.

    See also: How to view assembly dependencies?

  • Import

    • This last item of the `Tool' menu (as well as the following toolbar button:) gives you access to the import dialog, which enables you to import signed assemblies to the Eiffel assembly cache.

    See also: How to import a signed assembly to the Eiffel assembly cache?

The `Help' menu

  • Help Topics

    • This item (as well as the toolbar button ) opens the help topics about ISE Eiffel Studio where you can find the current help about ISE Assembly Manager.

  • About ISE Assembly Manager

    • This menu item gives you some general information about ISE Assembly Manager and the company Interactive Software Engineering, Inc.
