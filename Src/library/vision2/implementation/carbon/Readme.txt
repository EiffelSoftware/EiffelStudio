Vision4Mac
==========

Introduction
------------
The Carbon implementation of Vision2 (aka Vision4Mac) was started as a
semester project by four students from the ETH Zurich after the announcement
that EiffelSoftware was releasing EiffelStudio as Open Source.

(c) 2006-2007: Jann Roeder, Roland Haeusler, Ueli Peter, Daniel Furrer
