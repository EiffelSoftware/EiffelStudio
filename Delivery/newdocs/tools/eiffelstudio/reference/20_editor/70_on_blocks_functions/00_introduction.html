Introduction

Editing the text of Eiffel classes sometimes implies highly repetitive tasks, such as commenting or indenting several lines at the same time.

The new commands of the editor of EiffelStudio should allow the user to perform these tasks faster and more easily. In addition to text block commenting and indenting , EiffelStudio editor makes it possible with one click to embed several lines of code in "if ... then ... end" or "debug ... end" structures.
