Tutorial Step 1

This sample shows you how to set up your first WEL_APPLICATION which consists of a WEL_FRAME_WINDOW. This is one of the simplest WEL applications you can write, and the output is shown below:



Compiling

To compile the example:

  • Launch [BENCH].
  • Select Use existing Ace (control file) and click OK.
  • Browse to Eiffel50\examples\wel\tutorial\step1\.
  • Choose Ace.ace
  • Choose the directory where the project will be compiled, by default the same directory containing the Ace file.
  • Click OK.

Running

After you launch the sample, You should see the window displayed in the screen as illustrated above. You will have full control over the window, and the program will quit when you close the window.

Note: The size and shape of the window may be different on your machine.

Under the Hood

Although this system does nothing except create and display a WEL_FRAME_WINDOW, it shows how simple it is to set up your first WEL_APPLICATION.

This sample only contains one class, TUTORIAL_STEP1 which inherits WEL_APPLICATION and redefines main_window
