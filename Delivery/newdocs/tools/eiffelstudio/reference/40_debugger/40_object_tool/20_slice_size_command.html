Set slice size command

Located in the object tool toolbar, this command changes the display size of special objects (ie objects whose class inherits from SPECIAL). Since special objects may contain thousands of attributes, only the 50 first are displayed by default. If left-clicked, a dialog is popped up that sets the exploration indices of special objects that will be loaded in the future.

This dialog changes the settings for future special objects:

SCREENSHOT(dialog box set slice, global)

If a special object is dropped onto this command, the special object that is dropped is shrunk or extended in the tree according to the new exploration indices it is given in the dialog box. The dropped object may be any object, top-level or not.

This dialog changes the settings for a single special object:

SCREENSHOT(dialog box set slice, relative to an object)
