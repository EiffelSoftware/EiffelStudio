Gobo Eiffel Image

Gobo Eiffel Image, or geimage for short, is an open source tool which is able to embed an image into an Eiffel class which can then be used in applications using EiffelVision2. The image will then be compiled into the executable, so that there is no need to deploy the image file along side the executable anymore.
