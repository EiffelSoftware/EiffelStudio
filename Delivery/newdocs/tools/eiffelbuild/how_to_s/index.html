EiffelBuild How To's

How do I undo a mistake?

A full history is recorded by EiffelBuild as you work. There are three toolbar buttons located on the main toolbar which allow you to navigate the history. Modifictions to objects are not save in the history, but addition, deletion, and type changing of objects is recorded. Name changes for each object are also recorded.

How can I build using a visual representation?

Make the Builder window (Accessible through the main toolbar or menus) visible. This shows an approximate representation of your current project. Each of the object representations in this window may be used in the same way as the object representations in the Layout Constructor. For example, you may Pick an object from within the Layout Constructor and Drop it within a non full container representation in the builder window, then it will be moved acoordingly, and both views will update to reflect this change.

How can I see an accurate representation of my project?

Make the display window visible (On the main toolbar). This shows an accurate representation of the project you are building. This window is non-responsive to events, and unlike the builder window, it does not make containers visible through the use of a frame. Object modifications such as minimum sizes will not be reflected in the builder window, but in this display window, every property of an object that you modify will be reflected directly.

How can I insert an object at a paticular position within a container?

When you drop an object on another representation of a Vision2 container, it is always extended to the end. If you wish to insert the object at a paticular position, hold down shift while dropping, on the object contained in the container which you wish for it to be inserted before.

How do I change the type of an object with children?

Simply Pick from an existing object, and drop on the type of the object which you wish to replace it with in the Type Selector. This will replace the exising object with a new one of the selected type. All the children of the existing object will be inserted into the new object. Note that if the object you are changing the type of has one or more children, you may only replace it with type that allows that many children.

How do I quickly display an Object editor

Ctrl right click on a representation of the object (Not in the display window though). This will create a new "floating" object editor, with the object targeted inside it. You may have as many object editors existing concurrently editing the same object if you wish. A change in one will be reflected in all.

How do I see what a component looks like

Make the Component Viewer visible which can be done from the main toolbar. Then either Pick and drop on the component icon in the Component Viewer, or Ctrl right click on the component you wish to view. A representation of the component will be displayed in the Component Viewer, and you can see if it is what you require. Both a "display" and "builder" view are available by selecting the appropriate icon from the toolbar within the component Viewer.

How do I change the name of the window class generated?

Open up the project settings, and there will be an available option for changing both the file name and the actual class name to be generated.
