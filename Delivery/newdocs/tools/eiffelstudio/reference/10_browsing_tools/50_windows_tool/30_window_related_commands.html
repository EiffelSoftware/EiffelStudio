Window related commands

A few commands give the possibility to manage the development windows globally. They are located both in the Window menu and in the main toolbar.

  • New window: this command ( ) creates a new empty development window. Note that it is also possible to drop a class or a feature on it to create a development window that is immediately centered on the dropped item.
  • Minimize all: this command ( ) iconifies all development windows.
  • Raise all: this command ( ) raises all minimized development windows.
  • Raise unsaved windows: this command raises all the development windows that have not been saved. It may be combined with the Minimize all command to find easily which development windows need to be saved.
