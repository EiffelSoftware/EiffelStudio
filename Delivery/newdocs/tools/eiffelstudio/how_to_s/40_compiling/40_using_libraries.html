Using libraries

There are two kinds of libraries in Eiffel: precompiled Eiffel libraries and C libraries.

To use an Eiffel precompiled library:

  • Open the System Settings dialog.
  • In the General tab, enter the location of the precompiled library you want to use in the Precompiled Library field.
  • Click Apply or OK.
  • You must recompile your project for the changes to take effect.

SCREENSHOT(profiling in project settings dialog).

Note: You can only use one precompiled library at a time in a project. To use more than one, you should precompile all the libraries you want to use together in a single precompiled library.

To use a C library:

  • Open the System Settings dialog.
  • In the Externals tab, in the Include path field, type the paths of the header files needed to use the library (pressing the Add button for each).
  • In the Object file field, type the object files needed to use the library (pressing the Add button for each). These files can be either library files (.lib, .a, etc) or object files (.obj, .o, etc)
  • Click Apply or OK.
  • You must recompile your project for the changes to take effect.
