Running

After launching the application, you will see a window displayed with an appearance similar to the one above. On the left of the form you have the WINFORMS_TREE_VIEW control, and on the righ you have some controls to configure the WINFORMS_TREE_VIEW.

Under the Hood

The application shows how to use a WINFORMS_TREE_VIEW and what are all its different configurations.

This sample uses the following classes:

  • WINFORMS_TREE_VIEW
  • WINFORMS_GROUP_BOX
  • WINFORMS_CHECK_BOX
  • WINFORMS_COMBO_BOX
  • WINFORMS_LABEL

Note: This sample is translated from the example located in the QuickStart\winforms\samples\ControlReference\TreeView subdirectory of the .NET Framework SDK samples directory of Microsoft Visual Studio .NET.
