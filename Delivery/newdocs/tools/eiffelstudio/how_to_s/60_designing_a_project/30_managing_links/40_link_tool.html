Using the link tool

If you do not want straight arrows, you can insert handles on them. It is recommended that you use the link tool to do this, because it inserts the handles so that the angles in the line are exactly 90 degrees.

  1. To apply straight angles to a link, drop it in the Put handles on a link hole.
  2. The Link tool appears. Select the type of angle(s) you want for the link.

    The combinations of handles you can apply to your link are:
    • Put handle left
    • Put handle right
    • Put two handles left
    • Put two handles right
    • Put three handles left
    • Put three handles right
    • Remove handles
  3. Click Apply to see a preview the new layout of the link.
  4. Click OK when you are satisfied with the result, pressing Cancel will put the link back the way it was.
