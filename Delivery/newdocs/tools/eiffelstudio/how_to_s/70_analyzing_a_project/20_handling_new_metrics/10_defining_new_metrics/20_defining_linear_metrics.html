Defining linear metrics

To define a linear metric, you must have already clicked on of the "Metric" tab in the context tool. Select then the "Linear" tab of the "New metric" dialog:

  1. Enter a name in the "New metric name" field.

  2. Define a single term by entering a coefficient in the "Coefficient" field and choosing a metric from the "Metric" combo box. The unit field and the formula field will be automatically filled regarding the selected metric.

  3. Click on to validate the defined term. It will be displayed in the "Linear sum" field.

  4. To add other terms, repeat operation 2. and 3. as many times as needed.

  5. To remove the last entered term, click on . To remove a term, click on the same button as many times as needed, then re-enter missing terms.

  6. If you want to save the metric definition without exiting the interface, click on "Save", otherwise, click on "OK" to save and exit.

  7. If you want to define another linear metric, click on  to reset.

 

Note: After you have selected the first term of the linear sum, you can only add terms containing metrics of the same unit.
