Defining derived metrics.

To define a derived metric, you must have already clicked on of the "Metric" tab in the context tool. Select then the "Derived" tab of the "New metric" dialog:

  1. Enter a name in the "New metric name" field.

  2. Select a raw metric. Regarding the selected metric, you will get one of the following panel in the "Criteria" frame:






  3. Choose your criteria by selecting one button from each line.
  4. When you are evaluating this metric on a scope, all criteria could be met, or just some or none of them. If you want the currently defined metric to count scopes that have all the criteria select "All of the following are met", and if you want it to count scopes that have a t least one of the defined criteria, select "At least one of the following is met.
  5. If you want to save the metric definition without exiting the interface, click on "Save", otherwise, click on "OK" to save and exit.
  6. If you want to define another derived metric, click on  to reset

Note: Unit is adjusted in the "Unit" field regarding the selected raw metric
