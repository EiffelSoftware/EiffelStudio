The EiffelCOM library includes the following clusters:

  • A Constants cluster provides standard COM contants and enumerations.
  • A Interfaces cluster includes wrappers for standard interfaces that the EiffelCOM wizard does not generate and deferred classes: ECOM_INTERFACE, ECOM_QUERIABLE, and ECOM_STUB.
  • A Structures cluster includes wrappers for COM structures and additional data structures.
  • A Support cluster provides access to helper classes.
