Resource Bench features

The Eiffel Resource Bench:
  • Adds to the power of ISE Eiffel the ability to reuse the output of popular Interface Builders on Windows.

  • Parses any Windows resource file (as created by Microsoft Developer Studio or Borland Resource Workshop).

  • Shows resources through a "tree view" control.

  • Displays resource information.

  • Supports Open and Save of projects.

  • Generates both Eiffel Classes for producing an Eiffel graphical application (based on WEL, the Windows Eiffel Library) and an updated resource file.

  • Lets you specify which parts of your resource file you want to include in the code generation.

  • The perfect tool for Windows developer who want to get their product to market before the competition -- and with a better result.
