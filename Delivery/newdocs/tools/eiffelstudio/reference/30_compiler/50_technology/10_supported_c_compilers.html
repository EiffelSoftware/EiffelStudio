Supported C compilers

Unix/Linux Users:

ISE EiffelStudio supports `gcc' on most platforms and the native `cc' compiler if it is an ANSI C compiler.

 

Microsoft Visual C++ Users

ISE EiffelStudio supports only version 6.0 of the Microsoft Visual C++ environment. You can still use older versions of the Microsoft C++ environment (version 4.1, 4.2 or 5.0) as long as you have the latest Platform SDK from Microsoft, but this won't be supported by ISE.

Make sure that the Microsoft tools are correctly setup to run from the MS-DOS prompt:

  • On Windows NT or 2000, the installation of VC++ should do it automatically.
  • On Windows 95/98/Me, you need to copy the content of the file `vcvars32.bat' from the `bin' directory of the VC installation into your autoexec.bat file and to reboot your machine so that changes take effect.

By default the installation program will set in the registry keys the ISE_C_COMPILER key to `msc'.

 

Borland C++ Users:

ISE EiffelStudio only supports version 5.5.1 of Borland C++ that is included in the installation of ISE EiffelStudio. We do not support any other versions. Check with ISE for the availability of newer version of the Borland C++ compiler.

By default the installation program will set in the registry keys the ISE_C_COMPILER key to `bcb'.

 

Changing your C compiler on Windows

 

You can do it in two different ways. The first one is using the registry keys of the Windows™ operating system, the second one modifies the environment variables.

 

When modifying the registry keys, simply create a `compiler.reg' file with the following content: 

REGEDIT4

[HKEY_LOCAL_MACHINE\Software\ISE\Eiffel52]

[HKEY_LOCAL_MACHINE\Software\ISE\Eiffel52\ec]
"ISE_C_COMPILER"="C_COMPILER"

[HKEY_LOCAL_MACHINE\Software\ISE\Eiffel52\finish_freezing]
"ISE_C_COMPILER"="C_COMPILER"

[HKEY_LOCAL_MACHINE\Software\ISE\Eiffel52\wizard]
"ISE_C_COMPILER"="C_COMPILER"

 

Where you can replace "C_COMPILER" by either:

  • msc for Microsoft Visual C++
  • bcb for Borland C++

Then execute the file and everything will be ready to use your new C compiler.

 

When modifying the environment variables, simply defines ISE_C_COMPILER to be either:

  • msc for Microsoft Visual C++
  • bcb for Borland C++
