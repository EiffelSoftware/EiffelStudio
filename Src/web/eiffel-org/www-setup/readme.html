on Linux / install.sh

To easily install latest EiffelStudio on linux system:

curl -sSL https://www.eiffel.org/setup/install.sh | bash

note: EiffelStudio will be installed in your current directory, you will find a eiffel_*.rc file to configure your environment

on Windows / install.bat

To easily install latest EiffelStudio on Windows system:

curl -sSL -o %TEMP%\get-es.bat https://www.eiffel.org/setup/install.bat && call %TEMP%\get-es.bat && del %TEMP%\get-es.bat

Or you can download the file https://www.eiffel.org/setup/install.bat and execute it in a dos console.

Note: EiffelStudio will be installed in your current directory, you will find a eiffel_*.bat file to configure your environment
Requirements: the installation script requires 7z.exe, curl.ex or (wget.exe).
WARNING: installing EiffelStudio using the .msi file is safer as it performs additional registration operations useful for specific tools.

Customization

You can customize the installation with the following environment variables:

  • ISE_CHANNEL: latest (default), nightly, custom major.minor.build (i.e 18.11.102592 if it exists)
  • ISE_PLATFORM: if not set, the script tries to find expected value
On Linux, you can also pass arguments to the script using .. | bash -s channel platform. For instance:
  • curl -sSL ... | bash -s nightly
  • curl -sSL ... | bash -s latest linux-x86-64
On Windows, you can also use similar argument in the get-es.bat nighlty.
