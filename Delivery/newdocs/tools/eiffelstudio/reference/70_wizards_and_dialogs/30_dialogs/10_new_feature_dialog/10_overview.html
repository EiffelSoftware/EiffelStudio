New feature dialog overview

There are two parts in the window. The upper part allows you to choose what type of feature you want to create, a procedure, a function or an attribute.

The rest of the window is used to set the characteritics of the new feature. Information you will be asked depends on the type you chose. For more information, click on one of the links below :
