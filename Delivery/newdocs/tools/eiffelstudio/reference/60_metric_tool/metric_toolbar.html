Metric toolbar

The toolbar of the metric tools contains the following fields:

  •  name: to enter a name when saving a measure.
  •  scope: a combo box to select a scope type.
  •  metric: a menu containing all the available metrics .
  •  unit: displays the selected metric's unit.

And the following buttons:

  • calculate: evaluate a metric over a scope.
  • add: add a measure to the multi column list and save.
  • delete: delete a set of measures from the multi column list.
  • management: allow reordering, deleting or editing metrics.
  • new metric: create a new composite metric.
  • show details: show details of a measure.
  • hide details: hide details of a measure.
  • archive: create, update or compare archive files.
