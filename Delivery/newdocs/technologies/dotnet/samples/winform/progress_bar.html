Running

After launching the application, you will see a window displayed with an appearance similar to the one above. On the left of the form you have the WINFORMS_PROGRESS_BAR control, and on the right you have some controls to configure the WINFORMS_PROGRESS_BAR.

Under the Hood

The application shows how to configure a WINFORMS_PROGRESS_BAR.

The feature on_load is redefined and launch a thread THREAD_START that simulates the progression of a process.

This sample uses the following classes:

  • WINFORMS_PROGRESS_BAR
  • WINFORMS_GROUP_BOX
  • WINFORMS_TRACK_BAR
  • WINFORMS_COMBO_BOX
  • WINFORMS_LABEL
  • THREAD_START

Note: This sample is translated from the example located in the QuickStart\winforms\samples\controlReference\ProgressBar subdirectory of the .NET Framework SDK samples directory of Microsoft Visual Studio .NET.
