Retrieving a project from an Ace file

If you have already designed a project and want to retrieve it, or if you want to open a project created by someone else (such as the examples of the libraries which are shipped with EiffelStudio), you should select Open existing Ace (control file). It is located in the startup dialog:

and in the new project dialog:

Tip: If the project you want to retrieve is already compiled, you should select Open compiled project instead. This will avoid having to recompile the project.

A dialog is then popped up that prompts for the ace file and the project location:

Indicate the Ace file (*.ace) that contains information concerning the project you want to retrieve, and choose the directory where this project should be compiled.
