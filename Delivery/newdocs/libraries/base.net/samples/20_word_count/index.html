The Word Count Sample

This sample consists of a command line word counter. It provides quantitative information about the number of lines, words, characters and bytes in a given file.

Compiling

To compile the example:

  • Launch EiffelBench.
  • Select Use existing Ace (control file) and click OK.
  • Browse to Eiffel50\examples\base\word_count\.
  • Choose Ace.mswin.ace on Windows platforms or Ace.unix.ace on Unix platforms.
  • Choose the directory where the project will be compiled, by default the same directory containing the Ace file.
  • Click OK.

Running

To run the word count example, you must first open a console window. Here is the normal usage of the word count example: 

Usage: WordCount [-a] [-o] [-f<output-pathname>] input-pathname...
	-? Show this usage information
	-a Word usage sorted alphabetically
	-o Word usage sorted by occurrence and then alphabetically
	-f Send output to specified pathname instead of the console

Let us assume that you want to run the word count test over a file named WordCountTest, containing the following text: 

first sentence
second sentence
third sentence

The plainest command line for that test is: 

Wordcount.exe c:\WordCountTest.txt

The following result appear in the same console: 

Lines   Words   Chars   Bytes   Pathname
    3       6      43      47   c:\WordCountTest.txt
-----   -----   -----   -----   ---------------------
    3       6      43      47   Total in all files

It is possible to have more precise information about the specified file using  the following options

1) Alphabetical sorting: -a

The command line for sorting words in alphabetical order is: 

Wordcount.exe -a c:\WordCountTest.txt

The displayed result is then the same as above with the following precision: 

Word usage sorted alphabetically (4 unique words)
1: first
1: second
3: sentence
1: third

2) Occurrence and alphabetical sorting: -o

The command line for sorting words according to their occurrences is: 

Wordcount.exe -o c:\WordCountTest.txt

The displayed result is then the same as above with the following precision: 

Word usage sorted alphabetically (4 unique words)
1: first
1: second
1: third
3: sentence

3) Sending results 

Let us assume that you want to store the results of the word count test in a file rather than displaying it in the main console, for e.g. WordCountResult.txt file.You have then to use the -f option: 

Wordcount.exe -f c:\WordCountResult.txt c:\WordCountTest.txt

No results is displayed in the main console, but they are all stored in the specified output file that is: WordCountResult.txt.
