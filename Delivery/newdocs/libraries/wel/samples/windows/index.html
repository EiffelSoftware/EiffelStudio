Windows Sample



Compiling

To compile the example:

  • Launch [BENCH].
  • Select Use existing Ace (control file) and click OK.
  • Browse to Eiffel50\examples\wel\windows\.
  • Choose Ace.ace
  • Choose the directory where the project will be compiled, by default the same directory containing the Ace file.
  • Click OK.

Running

After launching the program, a window will appear with a custom ON_OFF_CONTROL control displayed. Clicking on this control will cause it's state to change. If you select one of the options from the "Windows" menu, that type of window will be displayed.

Under the Hood

The ON_OFF_CONTROL inherits WEL_CONTROL_WINDOW and demonstrates how to build your controls. The classes MODAL and MODELESS inherit WEL_MODAL_DIALOG and WEL_MODELESS_DIALOG respectively.

This sample contains the following classes:

  • APPLICATION_IDS
  • MAIN_WINDOW
  • MODAL
  • MODELESS
  • ON_OFF_CONTROL
  • WINDOWS_DEMO
