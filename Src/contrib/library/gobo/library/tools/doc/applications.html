Applications using the Gobo Eiffel Tools Library

The Gobo Eiffel Tools Library is an Eiffel class library useful for writing tools that expect Eiffel souce code as input. Here are a few examples using this library:

Compilers and interpreters

gec

Lint-like tools

gelint

Test tools

getest
TestStudio (superseded by AutoTest)

Ctags-like tools

gutentag

Reflection generators

Erl-G

Preprocessing tools

SCOOP to SCOOPLI preprocessor
