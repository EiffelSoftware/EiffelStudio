Array

To create and manipulate one-dimensional arrays, use class ARRAY of the Kernel Library. Arrays are not primitive language elements; instead, they are handled through class ARRAY. This class is “normal” in the sense that it may be used just as any other class by client and descendant classes. It is also somewhat special, however, in that the Eiffel.NET compiler knows about it and uses this knowledge to generate efficient code for array operations.
To create an instance of ARRAY, use the creation instruction
create my_array.make (u)
where the arguments indicate the lower and upper bounds. These bounds will then be accessible as my_array.lower and my_array.upper. The number of items is my_array.count.
To access and change the item at index i in array a, you may use features item and put, as in
x := my_array.item (i)
my_array.put (i, new_value)
