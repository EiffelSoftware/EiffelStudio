Using threads

Eiffel supports multithreaded programs. The HYPERLINK(thread library) thread library provides ways to handle threads safely inside an Eiffel program. Make sure you HYPERLINK(editing/adding_a_component) add it to your project if you want to use threads.

To create a multithreaded program or library:

  • Open the System Settings dialog.
  • In the Advanced tab, check the multithreaded runtime option.
  • Click Apply or OK.
  • You must recompile your project for the changes to take effect.

SCREENSHOT(multithreaded runtime in project settings dialog).

Caution: Make sure the external and precompiled libraries you use have also been compiled with support for threads.
