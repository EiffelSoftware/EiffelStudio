Discardable dialog windows

Some dialog windows in EiffelStudio include a check box labelled "Do not show me again". If you select it, the dialog will not be shown the next time the same situation occur. The "dialogs" sub-category of "Graphical elements" in the preference lists the state those discadable dialog windows : a boolean preference is associated with each of them. If it is True, the window will be shown, else it will not.
