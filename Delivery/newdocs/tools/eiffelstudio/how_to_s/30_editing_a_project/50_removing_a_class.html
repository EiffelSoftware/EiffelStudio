Removing a class

To remove a class, you can:
  • Center the editor on the component, and click on  in the standard toolbar or select Remove current item in the Tools menu.
  • Drop a pebble of this component on  in the standard toolbar or in the Diagram tool toolbar.
Caution: Removing a class from the system implies the deletion of the corresponding file on disk.
Caution: Picking a feature and dropping it on the delete icon, or calling Remove current item as the editor is centered on a feature, will not remove the feature but the class to which it belongs.
