Under the Hood

This sample shows how to draw some text in a window, using different brushes, fonts, styles and colors.

This sample uses the following classes:

  • WINFORMS_FORM
  • DRAWING_BRUSH
  • DARWING_COLOR
  • DRAWING_RECTANGLE
  • DRAWING_STRING_FORMAT

Note: This sample is translated from the example located in the QuickStart\Winforms\Samples\Gdiplus subdirectory of the .NET Framework SDK samples directory of Microsoft Visual Studio .NET.
