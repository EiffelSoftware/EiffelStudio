Reflection system specification



ISE reflection system is composed of five components interacting with each other: `ISE.Reflection.Emitter.exe', `ISE.Reflection.EiffelComponents.dll', `ISE.Reflection.Support.dll', `ISE.Reflection.CodeGenerator.dll' and `ISE.Reflection.ReflectionInterface.dll'. The two entry points are `ISE.Reflection.Emitter.exe' and `ISE.Reflection.ReflectionInterface.dll':

  • The emitter enables you to generate Eiffel classes and XML files from a .NET assembly.
  • The ISE reflection interface is aimed at connecting .NET names with corresponding Eiffel names by using XML files. It is a kind of database where you put XML files describing .NET types and you retrieve a more user-friendly type description.

N.B. This document refers to classes by using the Eiffel syntax and gives the .NET name in brackets, except if the assembly is written in C#.
