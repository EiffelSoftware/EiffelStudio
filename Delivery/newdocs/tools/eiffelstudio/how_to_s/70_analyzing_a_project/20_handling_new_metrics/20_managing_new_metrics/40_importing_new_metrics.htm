Importing new metrics

To import a set of metrics from a file, you must have already clicked on in the "Metric" tab of the context tool.

 

  1. Click on the "Import..." button.
  2. A dialog allows you to select a file containing a set of metrics.
  3. After you have opened a file containing metrics, the following interface will pop up to help you making your selection:



    - On selecting a metric, its definition formula appears in the field "Formula of the selected metric", its unit appears in the "Unit of the selected metric" field..
    - To import metrics from the selected file to the current system, you can either click on "Add->", or pick and drop the selected metric from the "New metrics" list to the "Current metrics" one
    - To remove a metric you have mistakenly imported, you can either click on "<-Remove", or pick and drop the selected metric from the "Current metrics" list to the "New metrics" one.
    - You can also reorder metrics using the pick and drop facility inside each list.
  4. Click on "OK" to validate importation.
  5. If you have introduced some changes, you have to save again on the main interface.
  6. Click on "Exit" to hide the interface.
