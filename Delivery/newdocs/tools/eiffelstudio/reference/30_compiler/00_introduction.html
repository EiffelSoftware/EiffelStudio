Compiler

EiffelStudio naturally includes a powerful Eiffel compiler, which is at the core of the IDE. It supports the entire Eiffel language as defined in the third edition of Eiffel: The Language.

This compiler uses the melting ice technology (TM), which accounts for lightning-fast incremental recompilations, ideal during the development cycle. On the other hand, finalization provides a high level of optimization, which should be used for released programs.

This reference describes some advanced functionality of the ISE compiler, such as making C calls from Eiffel or the reverse.
