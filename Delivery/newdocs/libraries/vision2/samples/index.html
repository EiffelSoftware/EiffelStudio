EiffelVision 2 Samples

Vision2 ships with a number of samples provided to demonstrate different aspects of the library. If you are a new user then it is suggested that you first compile and run the Widgets sample. This will demonstrate both the appearence and behaviour of the available widgets.

The other samples provided are listed below:

  • Accelerator - Demonstrates the use of an EV_ACCELERATOR.
  • Cursor - Demonstrates how to modify the displayed EV_CURSOR for an EV_WIDGET.
  • Gauges - Demonstrates different types of EV_GAUGE.
  • Standard_dialogs - Demonstrates different types of EV_STANDARD_DIALOG.
  • Viewport - Demonstrates EV_VIEWPORT.
