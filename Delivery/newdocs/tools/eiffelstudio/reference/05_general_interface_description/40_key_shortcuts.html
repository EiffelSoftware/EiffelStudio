Key shortcuts

Many operations in EiffelStudio do not require the use of the mouse. The same effect can be achieved through the keyboard only.

This page gathers all the keyboard shortcuts available in the environment. They are sorted in several categories, depending on their domain of application.

Editor shortcuts

Key shortcut Purpose
- clipboard shortcuts -
Ctrl+C Copy : Copy selected text into the clipboard.
Ctrl+Ins Equivalent to Ctrl+C.
Ctrl+X Cut: Copy selected text into the clipboard and remove it.
Shift+Del Equivalent to Ctrl+X.
Ctrl+V Paste: Insert clipboard content into the text.
Shift+Ins Equivalent to Ctrl+V.
- undo/redo shortcuts -
Ctrl+Z Undo last action.
Ctrl+Y Redo last undone action.
- search shortcuts -
Ctrl+F Display the Search tool and give the focus to the "Search for..." text field.
This shortcut may be customized.
Ctrl+H Display the Search tool and give the focus to the "Replace with..." text field.
This shortcut may be customized.
F3 Search the next occurrence of currently selected text or, if there is no selection, of the last searched expression.
This shortcut may be customized.
Shift+F3 Search the previous occurrence the last searched expression.
This shortcut may be customized.
- shortcuts to modify selected text -
Tab Indent selection (only if several lines are selected).
Shift+Tab Unindent selected lines or current line if there is no selection.
Ctrl+K Comment selected lines or current line if there is no selection.
Ctrl+Shift+K Uncomment selected line or current line if there is no selection.
A line is uncommented only if '--' is at its beginning.
Ctrl+I Embed selection (or current line if there is no selection) in an "if...then...end" structure.
Ctrl+D Embed selection (or current line if there is no selection) in an "debug...end" structure.
- other shortcuts -
Ctrl+A Select the entire text.
Ctrl+Space Auto-complete current word.
This shortcut may be customized.

Note: These shortcuts are not available everywhere. All operations that do not imply changes in the text are available both in the editor and in the class and feature views inside the context tool. Other operations are available only in the editor.

Compiler

Key shortcut Purpose
F7 Melt the system. This is the standard way to compile a debuggable system.
Ctrl+F7 Freeze the system. This melts the system and recompile the generated C code.
Ctrl+Shift+F7 Finalize the system. This generates an optimized executable which is not debuggable.

Debugger

Key shortcut Purpose
F5 Run. This is the standard way to run a system and debug it.
Ctrl+F5 Run without breakpoint. Same as run except that breakpoints are ignored.
Shift+F5 Kill the debugged application
Ctrl+Shift+F5 Pause the debugged application.
F11 Step into.
F10 Step by step.
Shift+F11 Step out.

Miscellaneous

Key shortcut Purpose
- File shortcuts -
Ctrl+S Save the file currently edited in the editor.
Ctrl+O Open a class file.
Ctrl+N Open a new window.
- Editor format shortcuts -
Ctrl+Shift+T Display the text form in the editor.
Ctrl+Shift+C Display the clickable form in the editor.
Ctrl+Shift+F Display the flat form in the editor.
Ctrl+Shift+O Display the contract form in the editor.
Ctrl+Shift+I Display the interface form in the editor.
