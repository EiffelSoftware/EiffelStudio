Apart from using Eiffel compilers, there is no real equivalent of gelint.

ISE's Eiffel Inspector goes beyond the scope of gelint and detects suspicious code usages in otherwise valid Eiffel code.

The definitions of the Eiffel validity rules reported by gelint can be found in the ECMA Eiffel standard. It is also available as an ISO standard.
