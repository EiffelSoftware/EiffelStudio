Generation Tab

 

This tab contains information about the available code generation options.

Here is the appearence of the Generation tab with default settings for a Build project:

The options available in this tab are as follows:

  • Attribute declarations grouped

    • Should different objects of same type be declared individually, or grouped?

    button1: EV_BUTTON
    button2: EV_BUTTON
    button3: EV_BUTTON
    or
    button1, button2, button3: EV_BUTTON

  • Attributes declared as locals

    • Should attribute declarations be local to initialization feature, or attributes of the class?

  • Generate debugging information

    • If selected, then for every feature connected to an event, code will be generated which outputs to the console, the name of the feature, and the values of all arguments.

  • Client of EV_TITLED_WINDOW

    • If selected, then the generated window will be a client of EV_TITLED_WINDOW, otherwise it will be a direct descendent.
