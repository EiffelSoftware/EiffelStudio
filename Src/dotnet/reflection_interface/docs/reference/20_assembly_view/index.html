The assembly view


The assembly view lists the types of an imported assembly. The types appear in an table with both Eiffel and .NET names.
At the top of the window, the name, version, culture and public key of the selected assembly are repeated.
More than a list of types, the assembly view mainly aims at enabling you to edit types. In fact, a simple click on a type name (the Eiffel name) opens a new window, called type view, giving an editable short-form of the Eiffel class.
