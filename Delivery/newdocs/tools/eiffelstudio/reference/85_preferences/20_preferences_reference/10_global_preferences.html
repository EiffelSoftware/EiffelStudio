Global preferences

This categories gathers non graphical preferences that are not related to a tool in particular. These preferences are :
 
 
Label in the preferences window Complete Description
Command used to launch an external editor. You may open a class in an external editor by dropping the corresponding pebble on the external editor icon. This preference sets which command line will be used to launch the external editor. The parameter "$target" will be replaced in the command with the file name corresponding to the class, "$line" with the line number of the feature if you dropped a feature pebble, 1 otherwise.
Text files format. This boolean preferences sets whether .e files will be saved in DOS or UNIX format. If it is set to "True", files will be saved in DOS format, i.e. line separator will be made of a line feed and a carriage return. Otherwise, UNIX format will used, i.e. there will not be carriage returns.
Use external command to print ? This set whether you want to specify which command line will be used to print texts.
Command invoked to print text. This is the command used to print text if the previous preference, "Use external command to print?", is true. The parameter will represent "$target" the name of the file given to the external program (see next preference).
Send a postscript file to this command? If it is set to False, the command line above will be called on a basic text file containing what you want to print. Otherwise, the file will be a postscript file with a formatting close to the one on screen.
