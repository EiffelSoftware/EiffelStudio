Running

After launching the application, you will see a window displayed with an appearance similar to the one above.

Click on the WINFORMS_BUTTON "Click Me!" and a message box appears with the text entered in the WINFORMS_TEXT_FIELD.

Under the Hood

The application shows how to use the WINFORMS_MESSAGE_BOX control.

An event (WINFORMS_EVENT_HANDLER) is associated to a control (here to the WINFORMS_BUTTON). So the line my_button.add_click (create {EVENT_HANDLER}.make (Current, $on_my_button_clicked))

associates a click to my_button to the feature on_my_button_clicked. So every time the BUTTON my_button is clicked, the feature on_my_button_clicked is executed.

The feature my_button_clicked displays a WINFORMS_MESSAGE_BOX:

feature {WINFORMS_MESSAGE_BOX}.show (msg)

with the text contained in the TEXT_FIELD:

msg := my_text_field.text

This sample uses the following classes:

  • WINFORMS_FORM
  • WINFORMS_BUTTON
  • WINFORMS_TEXT_FIELD
  • WINFORMS_EVENT_HANDLER

Note: This sample is translated from the example located in the QuickStart\winforms\samples\accessible subdirectory of the .NET Framework SDK samples directory of Microsoft Visual Studio .NET.
