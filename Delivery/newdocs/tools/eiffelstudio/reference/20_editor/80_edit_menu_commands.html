Edit menu commands

In the Edit menu, the following commands are available:

  • Undo: undo last change.
  • Redo: redo last undone change.
  • Cut: Cut selected text.
  • Copy: Copy selected text.
  • Paste: Paste text from the clipboard.
  • Select All: Select the entire text.
  • Find: Show and go to the search panel.
  • Replace: Show and go to the search panel with replace option activated.
  • Search:
    • Find next: Find the next occurrence of the last searched expression.
    • Find previous: Find the previous occurrence of the last searched expression.
    • Find selection: Find the next occurrence of the selected text.
