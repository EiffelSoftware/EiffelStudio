Generating executables

There are several ways to compile an executable with EiffelStudio.

Compilation modes include the following: melting, freezing and finalizing. To learn more about the compilation semantics in EiffelStudio, see the compiler reference.

All three commands are available in the Project menu, under the entries Melt, Freeze and Finalize.

The Melt command is also available in the Project toolbar .

To solve the errors that occur during the Eiffel compilation, use the compilation error wizard. For errors occurring during the C compilation, check that the installation is not corrupted (all environment variables should be set, especially ISE_COMPILER and ISE_PLATFORM), and refer to your C compiler documentation for more information.
