Texts display preferences

You may customize texts display in EiffelStudio by modifying preferences in the "Graphical elements" category and in its "Colors" sub-category.

"Graphical elements" contains two properties :

  • The font used in EiffelStudio editors ("All-purpose editor font"). You can change this by using a font selection window.
  • The tabulation size : This integer set to how many blank spaces a tabulation will be equivalent on screen. The default value is 4.
The "Color" sub-category gathers color preferences. By modifying them, you will be able to choose in which color the different elements detected by the syntactical analyzer (such as keywords, numbers, comments,...) will be displayed. Changes will be made via a color selection window.
