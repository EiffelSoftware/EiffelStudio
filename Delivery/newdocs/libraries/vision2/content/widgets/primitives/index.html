EiffelVision 2 Primitives

All vision2 primitives inherit EV_PRIMITIVE.

What is a primitive?

A primitive is a vision2 widget that may not contain other widgets. Primitives may be placed in containers, but no widgets may be contained in a primitive. Some examples of primitives are EV_BUTTON and EV_LABEL.

Features

All primitives inherit from EV_TOOLTIPABLE and therefore can have tooltips assigned to them. They all inherit many features from EV_WIDGET Each descendent of EV_PRIMITIVE will typically have many available features, specific to their type. For example, EV_SEPARATOR has no extra features, but EV_LABEL has features for modifying the current font and also the text to de displayed.

Sizing

As EV_PRIMIITVE is of type EV_WIDGET, the following facilites are provided for setting the minimum size : set_minimum_width, set_minimum_height and set_minimum_size.

Note:In vision2, there is no way to set the current size of the primitive. The current size is dependent its parent's size and its minimum_size.

Items

Although no primitive can contain another widget, certain primitives can contain items.
