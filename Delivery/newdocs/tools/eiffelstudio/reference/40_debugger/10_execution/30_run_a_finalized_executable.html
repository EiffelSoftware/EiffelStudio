Run a finalized executable

As opposed to other execution commands, this command launches an application HYPERLINK(compile in finalized mode) compiled in finalized mode. This has the same effect as running the application in the EIFGEN/F_code directory from a console prompt.

No debugging is possible with this command, but it lets you test the real speed of your program without any overhead.

It is accessible through the project toolbar ( ) or through the project menu.
