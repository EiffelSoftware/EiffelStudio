EV_CLIPBOARD

EV_CLIPBOARD provides facilites for interacting with the clipboard of the native operating system.

It is not possible to create an object of type EV_CLIPBOARD, but access is provided through the feature clipboard in EV_APPLICATION

Note:Although clipboards on some platforms provides specific facilites for manipulation of advanced data types, the clipboard within vision2 only provides facilities for dealing with plain text.

EV_CLIPBOARD provides the following basic features:

  • text - Queries the current text of the clipboard.
  • set_text - Assigns a text to the clipboard.
  • remove_text - Empties the clipboard.
