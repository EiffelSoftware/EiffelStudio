Project Settings Window

 

This is where global settings regarding your project are going to be made. In here you can:

  • specify clusters where to look for your classes
  • which assertions level you want to check
  • which debug clauses will be activated
  • command line arguments and current working directory when debugging your application
  • which C externals your project needs
  • compiler options for optimization
  • advanced options such as multithreading

 

The window is organized in six tabs.
