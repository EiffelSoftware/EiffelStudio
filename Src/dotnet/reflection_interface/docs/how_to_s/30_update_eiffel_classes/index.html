How to update Eiffel classes?


  1. To update the Eiffel classes corresponding to an imported assembly, your current screen has to be the imported assembly viewer, which is the window appearing when you launch ISE Assembly Manager.
    You are now ready to update the Eiffel sources:

  2. Click on the name of the assembly you want to update corresponding Eiffel classes (in the array).

  3. Then you have the choice between clicking on the toolbar button and selecting the menu item `Eiffel Generation' in the `Tools' menu. In both cases the Eiffel generation dialog will appear.

Note: If you want to generate the Eiffel classes for an assembly that is not in the Eiffel assembly cache, you just have to import it and ISE Assembly Manager will automatically generate the corresponding Eiffel classes in the repository: "$ISE_EIFFEL\library.net\assembly_name" (where $ISE_EIFFEL is the path to your Eiffel delivery and `assembly_name', the name of the selected assembly).

See: How to import a signed assembly to the Eiffel assembly cache?
