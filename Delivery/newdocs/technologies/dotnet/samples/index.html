Eiffel ENViSioN! Samples

A number of samples are provided with Eiffel ENViSioN!, designed to illustrate different aspects of the libraries, and how to reuse them in your projects. The following samples are available:

Console Sample

  • The Calculator Sample uses various classes from EiffelBase to implement a command line reverse Polish calculator.

Threads Samples

  • The Pools Sample uses a thread pool to perform asynchronous operations.
  • The Timers sample uses timers to to diplay messages on the console at regular intervals.

ADO.NET Sample

  • The ADO.NET Sample uses ADO.NET to access the Northwind database and display the content of the Customers table on the console.

Windows Form Samples

The following Windows Forms samples are all taken from the Framework SDK Quickstart. They demonstrate various aspects of the Windows Forms library reused from Eiffel.

The samples have been ordered from easiest to most complex, it is therefore recommended that you run them in the following order:

  • The Hello World Form Sample is the simplest of all and the recommended starting point.
  • The Message Box Sample sample uses a button coupled to an event handler which displays a message box on a click event.
  • The Calendar and Dates Sample sample uses a DATE_TIME_PICKER control to display a calendar in a form.
  • The Progress Bar Sample sample demonstrates how to use various progress report controls.
  • The Menus Sample illustrates adding menus to a form.
  • The MDI Sample uses a Multiple Document Interface form.
  • The Tree View Sample sample covers various scenarios involving a tree view control.
  • The Calc Sample consists of a simple graphical calculator.
  • The GDI+ Text Sample uses GDI+ to display texts in various ways.

EiffelVision 2 Samples

The following samples demonstrate the use the award winning multi platform EiffelVision 2 GUI library.

  • The Accelerator Sample associates keyboard shortcuts to event handlers.
  • The Cursor Sample uses a multi column list to display various mouse cursors that can be set.
  • The Gauges Sample uses various gauge (progress report) controls.
  • The Standard Dialogs Sample sample shows all the standard dialogs available in EiffelVision.
  • The Viewport Sample uses a viewport control to scroll a picture.
  • The Widgets Sample uses many controls in one widget.

Wel Samples

The Windows Eiffel Library - WEL samples consist of a tutorial and a fully functionaly card game xCell.
