Vision2 Kernel

The Kernel cluster contains many of the fundamental building blocks used in a Vision2 application. The following classes are contained in this cluster:

  • EV_ACCELERATOR - Used for providing keyboard shortcuts in vision2 applications.
  • EV_ANY - A common ancestor for all vision2 classes.
  • EV_APPLICATION -
  • EV_CLIPBOARD - Providing access to the clipboard of the native operating system.
  • EV_COLOR - A vision2 representation of a color.
  • EV_COORDINATES - A vision2 representation of a coordinate.
  • EV_CURSOR
  • EV_ENVIRONMENT - Provides information about the current operating system environment.
  • EV_FONT - A vision2 representation of a font.
  • EV_KEY - A vision2 representation of a key.
  • EV_RECTANGLE - A vision2 representation of a rectangle.
  • EV_SCREEN - Provides access to the screen.
  • EV_SIMPLE_APPLICATION
  • EV_TIMEOUT - Provides simple timing facilities.
