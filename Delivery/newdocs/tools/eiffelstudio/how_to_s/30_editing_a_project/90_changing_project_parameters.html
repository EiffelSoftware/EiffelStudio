Changing project parameters

A tool has been designed to help you to modify the settings of your project. HYPERLINK (project configuration) To make the Project configuration tool appear, there are two possibilities:
  • Click on  in the standard toolbar.
  • Select Project settings... in the Project menu.
This tool will allow you, for instance, to add or remove clusters, set the assertion level or activate debug clauses.

For more complete information, please refer to the Project configuration tool in EiffelStudio help.
