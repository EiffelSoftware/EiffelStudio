Creating an archive

Users may want to archive the result of the system they are working on. To create an archive, you must have already clicked on in the "Metric" tab of the context tool.

  1. In the "Create/replace archive" frame, click on "Browse".
  2. Select a folder and enter the name of the archive you are about to create. (Do not forget the ".xml" location!)

All the available metrics (except for scope comparison ones, since) are then evaluated over the system scope. Measures are saved in the specified file.

Note: A warning dialog will inform you in case an error occurs (the created file already exists, or you can not create a file in current location).
