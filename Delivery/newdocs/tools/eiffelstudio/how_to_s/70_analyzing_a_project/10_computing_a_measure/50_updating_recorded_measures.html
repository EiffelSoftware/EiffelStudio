Updating recorded measures

After recompilation, some of the saved measures are no longer accurate. They are pointed by the icon in the multi column list. It is possible to update the previously saved measures.

To update a result (or a set of result):

  1. Select a row or a set of rows (using SHIFT or CTRL keys) from multi column list.
  2. Right click on the multi column list.
  3. Choose from the pop up menu to update either selected rows or all rows.

Updated rows will be pointed by a green tick. New measures will also be updated in the file located in the "Metrics" subdirectory of current project.
