Introduction

There are a lot of properties of EiffelStudio that may be changed by the user. This section will present them, following the classification that can be found in the preferences window. If you open this window, you will see that properties are distributed in three categories :
  • Global preferences : This category contains non graphical preferences that are not specific to a tool in EiffelStudio.
  • Tools : This category gathers preferences associated to a particular tool in EiffelStudio.
  • Graphical elements : In this category, you will find the customizable properties of the graphical interface.
