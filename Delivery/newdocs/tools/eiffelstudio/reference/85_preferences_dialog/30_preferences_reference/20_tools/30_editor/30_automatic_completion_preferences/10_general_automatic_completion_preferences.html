General automatic completion preferences

You will find the following preferences in the "Automatic completion" subfolder of the "Editor" preferences folder :

  • Brackets and parenthesis are closed automatically: If this is set to True, a closing bracket will be automatically inserted when you type an opening one. It means that ')', '}' or ']' will be added respectively when you write '(', '{' or '['. If it is set to False, this feature is disabled. The default value is False.

  • Quotes are closed automatically: If this is set to True, '"', ''' or ''' will be added respectively when you type '"', ''' or '`'. If it is set to False, this feature is disabled. The default value is False.

  • Enable syntax autocomplete: If this set is to True, automatic syntax completion will be triggered when you type a keyword. If not, keywords will be handled the same way as other words. The default value is True.

  • Any features appear in the feature call complete window: If this is set to True, the features of class ANY will appear in the list of the feature call completion window. Otherwise, they will be hidden. The default value is False.
