Reordering new metrics

To reorder new metrics, you must have already clicked on in the "Metric" tab of the context tool.

You can then either use the "Up" and "Down" buttons, or use the pick and drop facility.

1) The "Up" and "Down" buttons

  1. Select a metric ("Up" and "down" are not sensitive when nothing is selected, neither if if the selected metric is respectively the first or the last one)
  2. To move up the metric, click on "Up", if sensitive, as many times as needed.
  3. To move down the metric, click on "Down", if sensitive, as many times as needed.
  4. Click on "Save" to validate your changes.
  5. Click on "Exit" to hide the dialog.

2) The pick and drop facility

  1. Select a metric by right clicking.
  2. Drop the selected metric on another one to insert it.
