How to update Eiffel classes?

  1. Go to the Imported Assembly Viewer
  2. Click on the name of the assembly corresponding to the external classes you want to update.
  3. Select Eiffel Generation from the Tools menu or click on the corresponding toolbar button .

Note: The Assembly Manager generates the external classes whenever an assembly is first imported. The update functionality should be used whenever these files have been corrupted to restore a valid repository. The default path for generated external classes is $ISE_EIFFEL\library.net\assembly_name where $ISE_EIFFEL is the path to the Eiffel delivery and assembly_name, the name of the assembly.
