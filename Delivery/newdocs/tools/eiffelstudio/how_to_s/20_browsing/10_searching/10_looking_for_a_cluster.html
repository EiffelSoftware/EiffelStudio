Looking for a cluster

If you know the name of a cluster but not where it is located in the system, there are two ways to find this out:
  • You can try to find it in the cluster tool by expanding the cluster tree.
  • In a window where the context tool is independent from the editor, you can type the name of the cluster in context tool address bar. You may use wildcards ("*" or "?") if you know only a part of the name. You can then locate the cluster in the system by picking the cluster stone in the address bar and dropping it on the view icon  of the cluster tool.
