EV_ANY

EV_ANY is the base class for the vision2 interface. Many of the features within this class are only exported to other vision2 classes and are used to ensure consistency throughout vision2.

The following features defined in EV_ANY are exported to ANY, thereby making them accessible to a vision2 application developer:

  • destroy - Destroys the underlying native toolkit object and renders the object unusable.
  • is_destroyed - Has the current object been destroyed?
  • copy - Update the fields of the current object with those of another object so as to yield equal objects. In vision2, in order for a class to be copyable, it must have redefined this feature. The following vision2 classes are the only ones which implement a redefinition and therefore are the only copyable vision2 classes: EV_FONT, EV_COLOR, EV_CURSOR and EV_PIXMAP.
