How to edit an imported assembly?


  1. To edit an imported assembly, your current screen has to be the imported assembly viewer, that is the window that appears when you launch ISE Assembly Manager.
    You are now ready to remove an assembly from the Eiffel assembly cache.

  2. Click on the name of the assembly you want to edit (in the array).

  3. Then you have the choice between clicking on the toolbar button and selecting the menu item `Edit' in the `Tools' menu. In both cases the assembly view will appear, displaying the list of types in the assembly. If you want to edit a particular type, just simple click on the type name (the Eiffel name) and the type view will be opened.
    This class short form is editable: it gives you the opportunity to rename the class name, the name of each feature and its arguments names.
