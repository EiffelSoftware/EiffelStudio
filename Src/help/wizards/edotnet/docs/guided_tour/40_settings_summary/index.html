Settings summary and generation dialog

The final dialog of the .NET wizard gives a summary of the application settings, that is the application type, project name, location, root class name, creation routine name and the list of the selected .NET assemblies.

Upon clicking on Finish, the .NET Wizard generates the project files including an empty root class. If Compiled generated project was checked in the Project Name and Location Dialog then the wizard also compiles the generated class.
