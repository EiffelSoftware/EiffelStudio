Vision2 Properties

This cluster contains all the common properties available for Vision2 widgets and items. Every Vision2 widget has the same set of properties inherited from EV_WIDGET, but many widgets also inherit additional properties, further refining the behaviour of the widget.

EV_WIDGET inherits the following properties:

  • EV_PICK_AND_DROPABLE
  • EV_SENSITIVE
  • EV_COLORIZABLE
  • EV_HELP_CONTEXTABLE
  • EV_POSITIONED
  • EV_CONTAINABLE

The following properties are also used within Vision2:

  • EV_DESELECTABLE
  • EV_DRAWABLE
  • EV_FONTABLE
  • EV_PIXMAPABLE
  • EV_POSITIONABLE
  • EV_SELECTABLE
  • EV_TEXTABLE
  • EV_TOOLTIPABLE
