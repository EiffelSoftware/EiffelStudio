Defining new metrics

The metric interface provides by default a set of basic metrics. However, user may want to define other metrics, such as linear sums or different ratios, or combining some criteria... This is possible thanks to the button of the context tool. An interface, "New metric" dialog, allows then the creation of four different kinds of metrics:

  • Derived metrics
  • Linear metrics
  • Ratio metrics
  • Scope comparison metrics

New metrics are increasingly added to the metric menu, and are available for calculation.
