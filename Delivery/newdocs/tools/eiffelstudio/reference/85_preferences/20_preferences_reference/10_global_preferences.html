Global preferences

This categories gathers non graphical preferences that are not related to a tool in particular. These preferences are :
 
 
Label in the preferences window Complete Description
Text files format. This boolean preference sets whether .e files will be saved in DOS or UNIX format. If it is set to "True", files will be saved in DOS format, i.e. line separator will be made of a line feed and a carriage return. Otherwise, UNIX format will used, i.e. there will not be carriage returns.
Command used to launch an external editor. You may open a class in an external editor by dropping the corresponding pebble on the external editor icon. This preference sets which command line will be used to launch the external editor. The parameter "$target" will be replaced in the command with the file name corresponding to the class, "$line" with the line number if this information is relevant, with 1 otherwise.
Use external command to print ? This sets whether you want to specify which command line will be used to print texts.
Command invoked to print a text. This is the command used to print a text if the previous preference, "Use external command to print?", is true. The parameter will replace "$printer" with the printer name on GTK platforms, and "$target" with the name of the file given to the external program (see next preference).
Send a postscript file to this command? If it is set to False, the command line above will be called on a basic text file containing what you want to print. Otherwise, the file will be a postscript file with a formatting close to the one on the screen.
Number of projects kept in the recent projects list This integral preference defines the maximum number of entries that should be displayed in the File/Recent projects menu.
Internet browser (GTK platforms only) This sets the command line that should be used to open an internet browser on a given HTML page. If present, "$url" will be replaced with the URL of the HTML page. This is used to display help files.
