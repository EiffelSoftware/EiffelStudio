The Gobo Eiffel String Formatter Library is based on the SiG Formatting Cluster of Eiffel/S (now Object Tools). SiG put the Formatting Cluster in the public domain by licensing it under the MIT License. It was ported by Friedrich Dominicus to other Eiffel compilers. The Gobo Eiffel String Formatter Library is based on Friedrich Dominicus's work, which can be downloaded from the Eiffel Zone page.
