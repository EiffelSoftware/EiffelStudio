Introduction

This section of the help deals with preferences related to a tool in particular.
These preferences are distributed in three categories :
  • Browsing tools preferences : They set some properties of the tools of the explorer bar (i.e. the left part of EiffelStudio development windows)
  • Context tools preferences : They will be used to change properties of some of the context tools.
  • Editor preferences : They will allow you to customize some features of the editor, automatic completion in particular.
