Assembly Selection Dialog

Selecting .NET assemblies

In this dialog, you can select the assemblies your application relies on.

The list of available assemblies appear at the top of the window. Any assembly which has been imported into the Eiffel Assembly Cache is available.
To add an assembly reference to your application, select the assembly name and click on Add: the selected assembly will moved to the list of selected assemblies.

Importing .NET assemblies

Importing signed assemblies

To import new signed assemblies in the list of available assemblies, click on ISE Assembly Manager.
For more information about the Assembly Manager, see the corresponding help topics.

The list of selected assemblies is updated upon closing the Assembly Manager: new assemblies available in the Eiffel Assembly Cache are automatically added to the list.

Importing local assemblies

To import non-signed assemblies locally, click on Import Local Assemblies. The following screen appears:

  • Select the non-signed assembly you want to import by typing in the corresponding text field and/or browse for a .NET Assembly by clicking on Browse....

  • Choose a directory where the Eiffel classes should be generated. By default, the assembly will be imported in the directory $ISE_EIFFEL\library.net (where $ISE_EIFFEL is the path to your Eiffel delivery).

  • The option Generate Eiffel-friendly names is checked by default, i.e. the generated code will be in lower case with underscores as recommended in Eiffel. Nevertheless, you may want to keep the original .NET syntax by unchecking this option.

The list of selected assemblies is updated upon completing Eiffel code generation: the newly imported assembly is automatically added to the list (only the name of the assembly appears since it is a non-signed assembly).

Note: If you happen to choose a signed assembly, the .NET Wizard automatically imports the selected assembly and its dependencies into the Eiffel Assembly Cache and the new assemblies available are automatically added to the list of selected assemblies.


To remove an assembly from the selected list, select the assembly name and click Remove: the assembly name will be moved to the list of available assemblies.
