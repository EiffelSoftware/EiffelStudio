Project settings window

 

This is where global settings regarding your project are going to be made. In here you can specify:

  • The type of project you wish to generate
  • The system name
  • Class names
  • Code generation options

The project window can be raised by clicking on the main toolbar, or by selecting Settings... from the Project menu.

 

The window is organized as three tabs:

  1. General
  2. Build
  3. Generation

All options are enabled and saved into the project settings file, 'project_settings.xml' as soon as the window is closed.
