Setting the command line arguments

To debug a program that uses command line parameters, it is possible to have the debugger call the generated program with certain command line arguments automatically. 

To change the command line arguments used when debugging:

  • Open the System Settings dialog.
  • In the Debug/Profile tab, type the command line arguments you want to use when debugging in the Program arguments entry.
  • Click Apply or OK.
  • You must recompile your project for the changes to take effect.

Note: All command line arguments are automatically saved, which makes it easy to switch between different command lines.
To remove a command line definitely, click the delete button on the right of the Program arguments entry.

Tip: To get the command line arguments in a program, it is possible to either define the root feature as taking an array of STRING objects, or to use the HYPERLINK(base reference: ARGUMENTS class) ARGUMENTS class, which provides many functionalities linked with command line arguments.
