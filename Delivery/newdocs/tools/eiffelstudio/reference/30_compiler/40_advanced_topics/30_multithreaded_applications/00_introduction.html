Introduction

Multithreaded applications provide a flexible, exciting way of utilizing the power of modern computer systems. ISE Eiffel supports a powerful multithreaded model, fast and easy to use. The EiffelThread library is mapped to the C thread standard libraries: it supports Windows, POSIX-like and Unix International Threads.

This reference first describes the steps to compile a multithreaded system, and the semantics of once features in multithreaded applications. It then explains how to use the EiffelThread library. EiffelThread is supported by the ISE compiler and run-time on Windows, Linux, Solaris, SGI, VxWorks, LynxOS, CRAY and Unixware.
