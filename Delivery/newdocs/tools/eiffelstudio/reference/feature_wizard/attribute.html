Attribute

When clicking Attribute, the wizard changes to the attribute layout. It has the following components:

  • Feature clause selection
  • Feature name field
  • Type selection
  • Header comment field
  • Invariant field
  • Set-procedure checkbox

Note: you can also click on a component in the image above to find out more.

Example

class
    PRODUCT

feature {NONE} -- Access

    price: DOUBLE
            -- Cost in dollars.

feature -- Element change

    set_price (a_price: DOUBLE) is
            -- Assign `a_price' to `price'.
        require
            a_price_non_negative: a_price >= 0.0
        do
            price := a_price
        ensure
            price_assigned: price = a_price
        end

invariant
    price_non_negative: price >= 0.0

end -- class PRODUCT
