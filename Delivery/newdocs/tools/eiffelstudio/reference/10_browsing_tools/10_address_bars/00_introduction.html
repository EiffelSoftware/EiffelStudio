Introduction

All development windows are composed of two major components, each of which can be centered on a given "address" (either a cluster, a class or a feature). One is the editor, with which go all the peripheral tools (including the cluster tool and the feature tool). The other is the context tool, which contains for example the HYPERLINK(formatters)formatters, the HYPERLINK(diagram tool)diagram tool and the HYPERLINK(metric tool)metric tool.

It is possible to either group both components so that they share the same address (allowing to have for instance formatter information on the class that is being edited immediately), or to separate them (allowing to have for instance formatter information on any class while editing a class).

When both are grouped, only the main toolbar appears, whereas when they are separated, the context tool address bar is also displayed, since it might point to a different location than the main toolbar.

The toggle data mode command allows you to switch between both modes.
