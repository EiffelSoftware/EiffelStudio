History functionality

EiffelStudio editor provides an history functionality, i.e. Undo and Redo commands.

  • To undo last change, press the key combination "Ctrl+Z", click on the undo icon or choose Undo in the Edit menu. If you repeat this operation, the editor will undo the before last change and so on until there is no more change to undo.

  • To redo last undone change, press the key combination "Ctrl+Y", click on the redo icon or choose Redo in the Edit menu. If you repeat this operation, the editor will redo the before last undone change and so on until there is no undone change left.
