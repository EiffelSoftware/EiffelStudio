Assembly Selection Dialog

Selecting .NET assemblies

In this dialog, you can select the assemblies your application relies on.

The list of available assemblies appear at the top of the window. Any assembly which has been imported into the Eiffel Assembly Cache is available.
To add an assembly reference to your application, select the assembly name and click on Add: the selected assembly will moved to the list of selected assemblies.

Importing .NET assemblies

To import new assemblies in the list of available assemblies, click on ISE Assembly Manager.
For more information about the Assembly Manager, see the corresponding help topics.

The list of selected assemblies is updated upon closing the Assembly Manager: new assemblies available in the Eiffel Assembly Cache are automatically added to the list.

To remove an assembly from the selected list, select the assembly name and click Remove: the assembly name will be moved to the list of available assemblies.
