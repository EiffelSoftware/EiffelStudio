Gelint Limitations

The following Eiffel constructs are not supported yet:

Eiffel for .Net
Only Eiffel classic systems are supported. It is possible to plug to gelint the code used by ISE EiffelStudio to read the .Net assemblies and generate AST of the corresponding Eiffel classes out of them. I did that a few years ago at work, and it worked. This is subject to ISE's licensing policy of course. Please contact me if you need some help.
