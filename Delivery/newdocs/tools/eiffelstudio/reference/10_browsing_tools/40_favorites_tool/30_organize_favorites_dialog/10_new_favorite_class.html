New favorite class command

Located in the Organize Favorites dialog, this command ( SCREENSHOT(Create new Favorite class button) ) provides a way of adding a class to the favorites. When clicked, a new dialog is popped up, that allows to select a class among the clusters of the system.

Clicking on a class name sets its name in the top text field, whereas double-clicking it automatically selects it and closes the dialog box. It is also possible to directly type the name of the wanted class in the top text field to select it.

The selected class goes to the root of the favorite tree. It is then possible to move it to a different favorite folder.
