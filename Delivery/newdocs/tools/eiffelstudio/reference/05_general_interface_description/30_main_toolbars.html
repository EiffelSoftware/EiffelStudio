Main toolbars

At the top of a development tool, two toolbars are displayed by default. They are separated by the address bar, which contains class and feature fields and format selection buttons. The standard toolbar is above the address bar, the project bar below :
SCREENSHOT (toolbars + address bar + arrows)

You can hide toolbars (or the address bar) by unselecting them in the Toolbars sub-menu of the View menu or in the contextual menu that you can make appear by right-clicking in the empty space on the right of the bars. If you want them to be displayed again, just select them in the Toolbars sub-menu of the View menu.
