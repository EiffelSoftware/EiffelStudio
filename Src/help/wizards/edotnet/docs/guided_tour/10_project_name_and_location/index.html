Project name and location dialog

By default the wizard creates an application called my_dotnet_application in the directory C:\my_projects\my_dotnet_application. If this folder does not exist, the wizard will automatically create it.

You can change these settings by typing in the corresponding text field and/or browse for a folder by clicking on Browse....

Uncheck the box Compile the generated project if you do not want the wizard to compile the code it generates.

Upon clicking on Next, the wizard checks the validity of the project location and displays an error message in case the path is not valid:
