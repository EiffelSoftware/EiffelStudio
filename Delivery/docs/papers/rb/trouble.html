Troubleshooting

If you have problems when you parse a resource file, you can first try to remove the grammar file (its name is "rb.gram"), and then re-launch Resource Bench. This will recreate the grammar file to allow parsing resource files.

On a Pentium 120 the process takes less than one minute.
