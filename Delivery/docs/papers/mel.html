MEL: O-O graphics for Motif

MEL, the Motif Eiffel Library, is an Eiffel encapsulation of the principal facilities of Motif, the windowing and graphical system for X Windows on Unix and VMS. Thanks to MEL you can develop powerful graphical applications for these platforms.

MEL can be used directly or through the EiffelVision library. Refer to the overall description of EiffelVision for a description of the graphical architecture.
