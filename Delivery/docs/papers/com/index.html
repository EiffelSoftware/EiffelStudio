Release Notes, Build 5
  1. Bug fixes:
    • Various compilation problems of the generated code are corrected.
    • Some restrictive preconditions and postconditions are removed.
    • Multiple conversion function problems are corrected.
    • The wizard correctly generates interface hierarchy for dual interfaces.
    • Automatic initialization of structures is added.
    • Support for access of Eiffel class attributes from COM is added.
    • Multiple memory management problems are corrected.
  2. New features:
    • Connection points are implemented on the server side.
  3. Improvements:
    • Increased speed of code generation.
  4. Known limitations:
    • Control containers are not implemented.
