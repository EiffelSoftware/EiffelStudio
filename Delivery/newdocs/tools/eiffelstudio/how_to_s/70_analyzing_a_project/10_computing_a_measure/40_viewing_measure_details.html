Viewing measure details

It is possible to have some details when evaluating a metric.

  1. Evaluate a metric over a scope.
  2. If the details button becomes sensitive, click on it to display details.
  3. If not, sorry, no details are available for this measure.
  4. To come back to the main interface click again on details .
