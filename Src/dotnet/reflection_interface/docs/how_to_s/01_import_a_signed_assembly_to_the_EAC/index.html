How to import a signed assembly into the Eiffel assembly cache?

  1. Make sure the assembly is signed.

  2. Go to the Imported Assembly Viewer.

  3. Select Import Tool from the Tools menu or click on the corresponding toolbar button .

  4.  
    • To import an assembly from the Global Assembly Cache:
      1. Click on the name of the shared assembly.
      2. Select Import in the Tools menu or click on the corresponding toolbar button .
    • To import a signed assembly which is in not in the Global Assembly Cache:
      1. Select Open Assembly in the File menu or click on the open toolbar button .
      2. Browse for the assembly and click Open.
