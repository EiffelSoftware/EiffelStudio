Browsing tools

This sub-categories gathers the preferences related to the tools in the left part of [BENCH] development window. There are four of them :
Label in the preferences window Complete description
Maximum number of items in history combo boxes This sets the maximum number of steps in browsing that will be recorded, i.e. the maximum number of times you will have the opportunity to press the back arrow. The default value is 10.
Show only one tool. If this is True, only one of the browsing tools (i.e. Features, Clusters, Search, Windows and Favorites tools) will be displayed at a time. If it is False several tools can be shown simultaneously. The default value is False.
Automatically expand the feature tree If this is True, the feature tree, which contains feature names sorted by feature clauses, will be expanded so that you can see all the features immadiately. If it is False, you will only see the feature clauses at first and have to expand them to reach feature names. The default value is True.
