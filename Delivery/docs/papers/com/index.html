Release Notes, Build 4
  1. Bug fixes:
    • Various compilation problems of the generated code are corrected.
    • Some restrictive preconditions are removed.
    • Multiple conversion function problems are corrected.
    • The wizard correctly recognizes [source] interfaces.
  2. New features:
    • Enhanced comment generation: the wizard generates link to a help file if known; for each function argument the comments specify whether it is [in] or [out]; for each optional argument the comments specify the default value.
    • For each interface two implementations are always generated: client implementation (simple wrapping) and server implementation, which requires user completion.
    • Call backs are supported: Simple call back (passing interface pointer) and connection points (supported only for clients (sinks)).
  3. Improvements:
    • Increased speed of code generation.
  4. Known limitations:
    • Control containers are not implemented.
    • Connection points are not supported on the server side.
