Changing class names and formal generics

  1. Start pick-and-drop on the class bubble of which you want to change the name or formal generics.
  2. Drop it in the Change class name and generics hole.
  3. The Class header dialog appears. You can enter a new name and/or formal generics.
  4. If you changed the name, choose between these options:
    • Change name locally: the name will be changed in the class declaration only. Any references to this one are now invalid.
    • Global search/replace of name in compiled classes: like the option above, but every reference to the class in compiled classes are also replaced with the new name.
    • Global search/replace of name in entire universe: like the option above, but references are replaced even in classes that are not in the system.
  5. Click OK.
