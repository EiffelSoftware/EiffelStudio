Argument list

For functions and procedures, you can build a formal argument list. You can do this by clicking on the New argument button as many times as you need arguments.

Note: even though the feature wizard always shows parentheses even is the list is empty, they are not generated because of Eiffel's command-query separation (CQS) principle.

Note: if you add arguments to a routine, it is not possible anymore to select routine type once.

For every attribute you selected, enter a name in the textbox and a type using the type selection.

The code that is generated:

set_name (s: STRING) is
    do
    end
