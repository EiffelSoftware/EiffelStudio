Installation instructions

Before installing the .NET components for Eiffel.NET you need the .NET Beta 2 Framework SDK installed on your computer. In the case you don't have it, the installation of the .NET components will fail during the installation of EiffelStudio.

You can download the latest SDK from Microsoft by clicking here.

Note: It is not necessary nor is it sufficient to install Visual Studio.NET Beta 2. You need the Framework SDK in order to use Eiffel.NET.
